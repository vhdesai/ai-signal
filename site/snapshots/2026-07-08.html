arxiv.org

“HalluSquatting” Attack Can Trick AI Coding Assistants Into Running Malicious Code
thehackernews.com
July 8, 2026
  • Researchers detailed HalluSquatting: identifying the fake package or tool names that AI coding assistants reliably hallucinate, registering them first, and waiting for an assistant to fetch the trap on a user's behalf.
  • Chaining a hallucination with a prompt injection, the technique led assistants to run attacker-supplied code — and a single popular planted name could reach many machines, effectively assembling a botnet.
  • It is a concrete supply-chain risk for any agentic developer tool that can fetch and execute external resources with limited human review.
