# GitHub Copilot shifts to token-based billing, drawing developer backlash

Microsoft is moving GitHub Copilot toward usage/token-based pricing, prompting developers on Reddit and X to warn of sharply higher costs — with some threatening to cancel. The shift mirrors Anthropic's Claude Code consumption model and reflects how the economics of agentic coding tools increasingly pass compute costs to end users. For organizations standardizing on AI coding assistants, the change reframes Copilot from a predictable flat-fee productivity tool into a metered operating expense that needs active cost governance.
