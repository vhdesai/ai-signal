# Google makes computer use a built-in tool in Gemini 3.5 Flash

Google folded "computer use" natively into Gemini 3.5 Flash, letting agents see, click, type and scroll across browser, mobile and desktop environments via the Gemini API and the renamed Gemini Enterprise Agent Platform — replacing the standalone Gemini 2.5 model. DeepMind paired it with targeted adversarial training for prompt injection plus two optional enterprise safeguards (explicit confirmation for sensitive actions; auto-stop on detected injection). For enterprises, it lowers the integration barrier for GUI automation but raises monitoring and reproducibility demands.
