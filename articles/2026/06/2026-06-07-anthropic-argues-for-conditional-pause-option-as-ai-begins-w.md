# Anthropic Argues for Conditional "Pause" Option as AI Begins Writing Its Own Code

Anthropic's Jack Clark and Marina Favaro argued that labs should preserve the option to slow or pause frontier development if "recursive self-improvement" approaches. Anthropic disclosed that as of May 2026, Claude authored more than 80% of code merged into its own codebase, with the typical engineer merging ~8× more code per day than in 2024. The proposed pause is explicitly conditional — contingent on multiple labs agreeing and verifiably stopping.
