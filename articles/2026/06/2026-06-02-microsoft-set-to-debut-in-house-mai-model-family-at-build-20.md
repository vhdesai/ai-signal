# Microsoft Set to Debut In-House MAI Model Family at Build 2026

Microsoft is expected to launch its homegrown MAI model family at Build today, including a coding model for the next-gen GitHub Copilot, alongside speech (MAI-Transcribe-1), voice, and image models. Early reporting indicates the coding model benchmarks at or above leading rivals on SWE-bench Verified at lower inference cost on Azure — Microsoft's most explicit signal of reducing dependence on OpenAI.
