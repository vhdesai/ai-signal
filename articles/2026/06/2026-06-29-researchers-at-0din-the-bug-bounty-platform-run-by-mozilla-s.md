# Researchers at 0DIN, the bug-bounty platform run by Mozilla's GenAI team, disclosed a technique to hijack developers'…

Researchers at 0DIN, the bug-bounty platform run by Mozilla's GenAI team, disclosed a technique to hijack developers' machines by getting an agentic coding tool to fetch and execute commands pulled via DNS from inside a GitHub repo — designed to evade standard scanning. The finding underscores emerging supply-chain and agentic-coding security risks as autonomous coding agents gain repo and shell access.
