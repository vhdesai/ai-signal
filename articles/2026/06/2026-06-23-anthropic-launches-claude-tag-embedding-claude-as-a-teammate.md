# Anthropic launches Claude Tag, embedding Claude as a teammate in Slack

Anthropic released Claude Tag in beta for Enterprise and Team customers, shifting Claude from a private chat box into shared Slack channels where any member can @-mention it to delegate asynchronous, multi-step work. Built on Opus 4.8, it persists context across channels, can operate “ambiently” to flag and follow up on work, and is scoped by administrators for data governance. Anthropic says 65% of its product team’s code now runs through an internal version — a shift from single-user chat toward shared, agentic enterprise workflows.
