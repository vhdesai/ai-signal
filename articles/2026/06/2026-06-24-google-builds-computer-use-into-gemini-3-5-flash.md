# Google builds "computer use" into Gemini 3.5 Flash

Google made computer use a native, built-in tool in Gemini 3.5 Flash, retiring the standalone Gemini 2.5 computer-use model and exposing the capability via the Gemini API and the renamed Gemini Enterprise Agent Platform. Agents can now see, reason about, and act across browser, mobile and desktop environments for long-horizon tasks like continuous software testing. Notably, Google paired it with targeted adversarial training for prompt injection plus two optional enterprise safeguards (explicit confirmation for sensitive actions; auto-stop on detected injection) — an explicit enterprise-trust pitch.
