# Microsoft Launches Scout Agent, Agent 365 Governance, and Surface RTX Spark Dev Box

Scout is the first “Autopilots”-class always-on agent, built on OpenClaw and grounded in Work IQ across M365. Each agent gets its own Entra identity; Agent 365 and Microsoft IQ provide governance. The Surface RTX Spark Dev Box delivers up to 1 petaflop with 128GB unified memory, running 120B-parameter models locally. Maia 200 is live in Iowa and Arizona data centers. Scout is available immediately to Frontier-program customers.
