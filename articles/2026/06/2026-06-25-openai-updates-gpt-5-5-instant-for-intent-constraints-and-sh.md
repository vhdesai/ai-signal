# OpenAI updates GPT-5.5 Instant for intent, constraints and shopping — already in the API

OpenAI shipped an update to GPT-5.5 Instant — the default model in free ChatGPT and its most-used model — improving multi-turn context, complex-constraint following, and shopping/local recommendations. The change is positioned as a move away from heavy prompt hand-holding toward better goal inference and is available immediately via the API. It is an incremental but high-volume update, notable because it lands amid reports that the larger GPT-5.6 release is being gated (see Policy). https://venturebeat.com/technology/openais-updated-gpt-5-5-instant-is-better-at-shopping-complex-constraints-and-understanding-user-intent-and-its-already-in-the-api
