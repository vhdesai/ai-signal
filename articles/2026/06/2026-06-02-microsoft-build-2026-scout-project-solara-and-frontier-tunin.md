# Microsoft Build 2026: Scout, Project Solara, and Frontier Tuning Define the “Agent-First” Era

Microsoft’s Build keynote delivered three headline product announcements. Scout is an always-on personal AI agent—described as OpenClaw-inspired—that operates across Windows, mobile, and web as a persistent digital coworker. Project Solara is an Android-based OS designed for AI agent gadgets—devices that run agents instead of apps—positioning Microsoft for a post-smartphone hardware category. Frontier Tuning allows enterprises to customize AI models on their own data without fine-tuning infrastructure. Collectively, the announcements reframe Microsoft’s AI strategy around agentic systems, not just copilots.
