# OpenAI Launches GPT-5.5-Cyber and "Patch the Planet" Bug Bounty

OpenAI released GPT-5.5-Cyber, a cybersecurity-specialized model, alongside a new "Patch the Planet" initiative that rewards researchers for AI-assisted vulnerability discovery. The program has already identified hundreds of real-world bugs. A companion Codex Security plugin integrates inline security scanning directly into developer workflows—positioning OpenAI as a serious enterprise security player, not just a model provider.
