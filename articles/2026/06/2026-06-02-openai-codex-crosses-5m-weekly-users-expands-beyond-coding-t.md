# OpenAI Codex Crosses 5M Weekly Users; Expands Beyond Coding to Knowledge Work

OpenAI published a knowledge-work report stating Codex now has over 5 million weekly active users — up 6× since February — with knowledge workers making up ~20% of users. OpenAI is repositioning Codex as a general-purpose autonomous work agent, not just a code assistant. If adoption holds, it represents a meaningful broadening of enterprise TAM beyond developers. ________________________________
