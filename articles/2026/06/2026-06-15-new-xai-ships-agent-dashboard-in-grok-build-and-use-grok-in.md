# New xAI ships Agent Dashboard in Grok Build and "Use Grok in Warp" integration

xAI posted two updates to its Grok Build agentic-coding product: an Agent Dashboard that lets developers manage many coding sessions at once, see what each is doing, reply to those that need attention, and dispatch new work; and a "Use Grok in Warp" integration. Both are incremental developer-tooling updates rather than model releases. The cadence underscores xAI's focus on agentic-coding workflows as its near-term competitive surface.
