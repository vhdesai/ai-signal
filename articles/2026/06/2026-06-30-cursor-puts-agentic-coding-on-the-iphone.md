# Cursor puts agentic coding on the iPhone

Cursor — the AI coding platform whose parent, Anysphere, agreed to a roughly $60B all-stock acquisition by SpaceX — released a public-beta iPhone and iPad app that lets paid users launch, monitor, review, and merge the work of cloud coding agents from mobile. Agents run in isolated cloud VMs and keep working with the app closed, surfacing progress via push notifications and lock-screen Live Activities. The release, which followed the app's June 29 debut, underscores a broader shift from writing code to supervising autonomous agents, joining mobile offerings from Anthropic and OpenAI.
