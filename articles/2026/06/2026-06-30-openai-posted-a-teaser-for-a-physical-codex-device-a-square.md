# OpenAI posted a teaser for a physical Codex device — a square, button-and-dial macro pad built with boutique keyboard…

OpenAI posted a teaser for a physical Codex device — a square, button-and-dial macro pad built with boutique keyboard maker Work Louder — arriving July 15 under the tagline "Your favorite Codex shortcuts are getting an upgrade." OpenAI clarified this is distinct from the more ambitious consumer AI device it is developing with Jony Ive's LoveFrom. The peripheral targets developer power users and hints at a nascent market for AI-native input hardware.
