# OpenAI Introduces "Dreaming" to Overhaul ChatGPT Memory

"Dreaming" processes and consolidates user context in the background — analogous to how sleep consolidates memory. OpenAI cut memory-synthesis compute by ~5×, making enhanced memory viable for free users. Launching first to Plus and Pro in the US.
