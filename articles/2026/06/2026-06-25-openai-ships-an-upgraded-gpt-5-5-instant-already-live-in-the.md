# OpenAI ships an upgraded GPT-5.5 Instant, already live in the API

OpenAI updated GPT-5.5 Instant — the default model for free ChatGPT users — with stronger intent inference, better handling of complex multi-constraint requests, and improved shopping and local-recommendation results. The company framed it as a move away from models that need heavy prompting toward systems that preserve user goals across turns. The revised model is already available to developers via the API.
