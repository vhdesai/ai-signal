# Anysphere's Cursor released Cursor Mobile, an iOS app that lets developers spin up and supervise autonomous coding…

Anysphere's Cursor released Cursor Mobile, an iOS app that lets developers spin up and supervise autonomous coding agents from their phones, extending the agent-centric direction of Cursor 2.0. The launch follows similar mobile moves from Anthropic and OpenAI and arrives weeks after SpaceX's reported ~$60B acquisition of the company. It reflects a broader shift in AI coding — from writing code to overseeing code-writing agents, increasingly untethered from the desktop.
