# ### Nvidia unveils RTX Spark superchip, reinventing Windows AI PCs with Microsoft

At GTC Taipei, Nvidia introduced the RTX Spark superchip—1 petaflop of AI compute and up to 128GB unified memory—built to run frontier models and personal agents locally on Windows laptops and desktops. The chip pairs a Blackwell RTX GPU with a 20-core Grace CPU (co-designed with MediaTek) and ships this fall via ASUS, Dell, HP, Lenovo, Microsoft Surface and others. Microsoft is contributing new Windows security primitives and an OpenShell runtime to run on-device agents safely.
