# Prompt injection named the top enterprise-AI attack vector as agents and RAG widen the surface

VentureBeat details how prompt injection — ranked LLM01 in the OWASP LLM Top 10 — has evolved beyond chatbots to target AI agents, RAG pipelines, model routers and long-term memory, with CrowdStrike documenting malicious-prompt injections across 90+ organizations in 2025 ("prompts are the new malware"). Newer variants include RAG supply-chain poisoning, agent hijacking, context-overflow and memory poisoning. As enterprises wire LLMs into workflows that send email and execute code, the piece argues models must be treated as untrusted interpreters, not trusted automation.
