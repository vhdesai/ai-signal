# DeepSeek released DSpark, an MIT-licensed speculative-decoding system that uses a lightweight "scout" to run a few…

DeepSeek released DSpark, an MIT-licensed speculative-decoding system that uses a lightweight "scout" to run a few steps ahead and guess likely next tokens, which the larger model then verifies — accelerating output by up to 85% without changing what the model says. VentureBeat notes the real-world speedup depends on how often the guesses are accepted, but the release continues DeepSeek's pattern of pushing the global cost-and-speed curve through open weights. Landing amid US restrictions on the latest Anthropic and OpenAI models, it reinforces China's open-source momentum as a competitive lever.
