# OpenAI Unveils Lockdown Mode to Protect Against Prompt Injection

Lockdown Mode disables live web browsing, web image retrieval, Deep Research, and Agent Mode — the surfaces most exploited by prompt-injection attacks. OpenAI is explicit it reduces, not eliminates, leakage risk. The release directly addresses agent-security vulnerabilities documented by Microsoft (7 attack vectors) and Anthropic (31.5% hijack rate) earlier this week.
