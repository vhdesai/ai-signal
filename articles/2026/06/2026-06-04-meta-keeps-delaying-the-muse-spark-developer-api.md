# Meta Keeps Delaying the Muse Spark Developer API

Meta's Muse Spark launched in April but the developer API has slipped repeatedly — close to two months — until Meta said this week it would ship "this month." Each delay intensifies scrutiny of Meta's AI spend relative to shipped, monetizable capability. A model without an API is a demo, not a platform.
