# The open-source agent project OpenClaw released iOS and Android companion apps that connect a phone to a user's…

The open-source agent project OpenClaw released iOS and Android companion apps that connect a phone to a user's self-hosted OpenClaw agent gateway running on a Mac, Windows machine, or server, adding voice input and per-task permissioning. The apps act as companions to an existing gateway rather than standalone agents.
