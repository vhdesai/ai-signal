# Princeton's CEO-Bench: only three models survive a 500-day startup simulation

Princeton researchers introduced CEO-Bench, a long-horizon agent test in which an AI must run a simulated software company for 500 days in a noisy, partially observable market with delayed, coupled consequences. Most current models go broke, only three finished above starting capital, and a simple rule-based heuristic with no AI beat nearly all of them. The result is a sober data point on agent reliability for autonomous, multi-step business operations — directly relevant to anyone scoping “AI employee” deployments.
