# VentureBeat details how prompt injection remains the top enterprise-AI attack vector (OWASP LLM01), targeting agents,…

VentureBeat details how prompt injection remains the top enterprise-AI attack vector (OWASP LLM01), targeting agents, RAG pipelines, and model routers. CrowdStrike's 2026 Global Threat Report found malicious prompt injection at 90+ organizations in 2025 ("prompts are the new malware"), with AI-enabled attack volume up 89% YoY — citing real incidents including EchoLeak, the first zero-click prompt-injection exploit, against Microsoft 365 Copilot.
