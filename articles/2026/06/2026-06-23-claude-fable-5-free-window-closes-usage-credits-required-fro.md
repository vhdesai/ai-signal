# Claude Fable 5 Free Window Closes; Usage Credits Required from June 23

Fable 5 exits its free-inclusion window today — usage now requires credits at ~$10/M input and $50/M output tokens (~2× Opus 4.8). No extension offered despite the model being offline for days during the window due to export-control action. Forces near-term cost and model-selection decisions for standardized teams.
