# OpenAI Unveils Lockdown Mode Against Prompt Injection

Lockdown Mode disables live web browsing, image retrieval, Deep Research, and Agent Mode — the surfaces most exploited by prompt injection. Reduces but doesn't eliminate leakage risk. Addresses agent-security vulnerabilities from Microsoft (7 vectors) and Anthropic (31.5% hijack rate).
