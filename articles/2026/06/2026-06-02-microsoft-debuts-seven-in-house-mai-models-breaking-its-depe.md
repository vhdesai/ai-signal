# Microsoft Debuts Seven In-House MAI Models, Breaking Its Dependence on OpenAI

Microsoft unveiled seven first-party “MAI” models, led by MAI-Thinking-1—a 35B active-parameter MoE reasoning model with 256K context—and MAI-Code-1-Flash, a 5B coding model rolling out in GitHub Copilot. Microsoft says MAI-Thinking-1 reaches parity with Claude Sonnet 4.6 and matches Opus 4.6 on SWE-Bench Pro while undercutting GPT-5.5 on cost. On-device “Aion” SLMs for Windows round out the family. The move marks Microsoft’s pivot from reselling OpenAI to participating at the frontier with its own models.
