# NVIDIA open-sourced a BioNeMo Agent Toolkit that wraps drug-discovery models—OpenFold3, DiffDock, and GenMol—as…

NVIDIA open-sourced a BioNeMo Agent Toolkit that wraps drug-discovery models—OpenFold3, DiffDock, and GenMol—as documented, callable "skills" for AI agents, describing each model's inputs, artifacts, and failure modes. In NVIDIA's benchmarks with Codex CLI and GPT-5.5, the skill layer raised task completion from 57.1% to 100% and roughly doubled token efficiency.
