# Amazon and OpenAI Build Stateful Model Runtime on Amazon Bedrock

Amazon and OpenAI announced a jointly built stateful runtime environment on Bedrock allowing applications to retain memory across conversations — critical for complex agentic workflows. Microsoft Azure retains exclusive rights to OpenAI's stateless APIs, making Amazon's stateful access uniquely differentiated. This is tied to Amazon's $50B OpenAI investment and OpenAI's $138B AWS cloud commitment, with OpenAI also consuming 2 gigawatts of Trainium capacity.
