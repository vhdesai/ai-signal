# Nvidia released Nemotron 3 Super under an open-source license, expanding its enterprise AI model portfolio

Nvidia released Nemotron 3 Super under an open-source license, expanding its enterprise AI model portfolio. The model is designed for instruction-following and enterprise reasoning tasks and is optimized to run efficiently on Nvidia hardware. The open release underscores Nvidia's dual strategy: selling compute infrastructure while simultaneously seeding the open-source model ecosystem to increase GPU demand.
