# Nvidia Launches DLSS 4.5 with Dynamic Multi Frame Generation — Up to 6x Performance

Nvidia released DLSS 4.5 today, introducing Dynamic Multi Frame Generation that intelligently shifts between frame multipliers to match display refresh rates up to 240Hz+. MFG 6x mode is available for RTX 50 Series. Beyond gaming, the technology demonstrates Nvidia's AI-driven rendering pipeline investment with growing relevance to simulation and synthetic data generation for AI training. 🛠️Products & Tools
