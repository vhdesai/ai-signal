# Anthropic Claude Gets Computer Use on Mac — Desktop Automation from iPhone

Anthropic's Computer Use feature — in research preview for Claude Pro and Max on macOS — allows Claude to autonomously control a user's desktop: clicking, typing, opening apps, and completing tasks remotely. The "Dispatch" companion lets users send instructions from their iPhone to be executed on their Mac. Claude uses a permission-first approach and prioritizes API connectors before screen control, directly competing with the viral OpenClaw agent platform.
