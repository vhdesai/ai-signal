# Anthropic Claude Code Source Leaked Again — Exposes "Capybara" Model Family

Security researcher Chaofan Shou found that Claude Code v2.1.88 contained a 57MB source map exposing 1,906+ proprietary TypeScript files — the second leak in a year. Analysis uncovered an unreleased "Capybara" model family (tiers: capybara, capybara-fast, capybara-fast-1m), frustration telemetry, and a hidden /buddy AI companion feature. Separately, Anthropic acknowledged quota limits hitting "way faster than expected," caused by prompt cache bugs inflating costs 10-20x — now the team's top priority.
