# Anthropic has launched computer-use capabilities inside Claude Cowork and Claude Code, allowing Claude to open…

Anthropic has launched computer-use capabilities inside Claude Cowork and Claude Code, allowing Claude to open applications, navigate browsers, fill spreadsheets, and execute entire workflows autonomously. The feature enters research preview on macOS for Claude Pro and Max subscribers, with Windows support planned. Users receive real-time notifications before critical actions and can monitor or review all steps. The launch represents a major step toward autonomous, device-level AI agents for knowledge workers.
