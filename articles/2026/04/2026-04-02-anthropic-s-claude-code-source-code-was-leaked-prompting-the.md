# Anthropic's Claude Code source code was leaked, prompting the company to take down thousands of GitHub repositories in…

Anthropic's Claude Code source code was leaked, prompting the company to take down thousands of GitHub repositories in an aggressive cleanup — a move Anthropic says was accidental. TechCrunch reported that within 24 hours of the leak, someone cracked the code signing system, raising concerns about the broader security implications. Anthropic acknowledged the takedowns were an error and is working to resolve the situation. The incident underscores growing security risks as AI development toolchains become increasingly open and complex.
