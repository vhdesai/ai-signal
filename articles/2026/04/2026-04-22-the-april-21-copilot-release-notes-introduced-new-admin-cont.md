# The April 21 Copilot release notes introduced new admin controls for AI video generation, a customizable Employee Self-Service agent landing page, and rich Bing interactive cards (weather, stocks) in Copilot Chat. Separately, Microsoft revealed its OneDrive 2026 roadmap — Copilot is now embedded directly in OneDrive for document summarization, PDF review, and file comparison. At Community Summit NA, Microsoft confirmed the Model Context Protocol (MCP) is now Generally Available across Copilot Studio, with Agent2Agent protocol as the next priority. Anthropic Claude Sonnet models are now on-by-default in Word, Excel, and PowerPoint.
