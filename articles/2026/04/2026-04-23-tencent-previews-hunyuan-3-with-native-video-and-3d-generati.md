# Tencent previews Hunyuan 3 with native video and 3D generation

Tencent previewed Hunyuan 3 (branded Hy3), emphasizing unified text, image, video, and 3D-asset generation from a single model. The company framed the release as infrastructure for game studios and advertising customers inside its ecosystem. Public API availability is expected in May.
