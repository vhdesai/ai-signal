# Cursor shipped a complete redesign centering the product around managing agents rather than editing files

Cursor shipped a complete redesign centering the product around managing agents rather than editing files. New features include a unified sidebar for local and cloud agent fleets, multi-repo workspaces, a built-in browser, and a plugin marketplace for one-click MCP/subagent installs. Agents initiated from mobile, web, desktop, Slack, GitHub, or Linear all converge in one view. The release is a direct bet that AI-native development means humans orchestrating fleets of agents. This comes as Claude Code reportedly holds 54% of the AI coding market per Menlo Ventures.
