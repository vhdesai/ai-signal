# OpenAI launches GPT-5.5 with step-change agentic coding performance

GPT-5.5 posts 82.7% on Terminal-Bench 2.0, a roughly 12-point lift over GPT-5, with notable gains on long-horizon software engineering tasks and tool use. OpenAI is pricing the model aggressively for API customers and rolling it into ChatGPT Enterprise this week. Early partner quotes emphasize multi-hour autonomous coding runs as the headline unlock.
