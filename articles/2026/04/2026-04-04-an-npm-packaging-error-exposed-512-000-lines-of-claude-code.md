# An npm packaging error exposed 512,000+ lines of Claude Code TypeScript source code, which spread to 84,000+ GitHub…

An npm packaging error exposed 512,000+ lines of Claude Code TypeScript source code, which spread to 84,000+ GitHub stars before DMCA takedowns. No customer data was exposed, but the leak revealed significant roadmap items: KAIROS—a persistent background agent operating autonomously even when Claude Code is closed; AutoDream—a memory consolidation system active during idle periods; and Undercover Mode—for stealth open-source contributions without Anthropic attribution. Security researchers have since identified malicious npm packages exploiting the leak to distribute malware targeting developer machines.
