# Microsoft & OpenAI restructure exclusive cloud deal

Microsoft and OpenAI restructured their partnership, ending cloud exclusivity while keeping Azure as OpenAI's primary cloud partner and removing AGI-linked deal terms. Direct context for today's "Deployment Company" raise — OpenAI now has both the contractual room and the capital to expand its enterprise distribution beyond Microsoft.
