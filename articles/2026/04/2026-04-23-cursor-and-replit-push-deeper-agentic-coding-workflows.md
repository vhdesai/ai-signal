# Cursor and Replit push deeper agentic coding workflows

Cursor shipped a “background agents” feature that lets engineers dispatch multi-hour coding jobs and review diffs asynchronously. Replit announced pricing changes for its Agent 3 product and new enterprise guardrails. Both moves reinforce the shift from completion-style assistants toward autonomous, managed coding agents.
