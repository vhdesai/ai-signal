# The HKUDS research group released RAG-Anything, an open-source "all-in-one" framework for Retrieval-Augmented Generation designed to work across varied data types and deployment contexts. The project aims to make RAG pipelines more accessible to developers and researchers who need to integrate external knowledge into large language models without building custom retrieval infrastructure from scratch. It is hosted on GitHub and attracting rapid interest from the developer community.
