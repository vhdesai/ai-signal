# OpenAI continued rolling out GPT-5.4 with significant gains on coding benchmarks (SWE-Bench Pro: 74.2%) and extended reasoning tasks, while announcing a sunset timeline for GPT-4o. The Codex CLI has been updated with GPT-5.4 as the default backend for agentic terminal-based coding workflows. OpenAI also introduced a new $100/month Pro plan tier targeted at high-intensity coding users running long autonomous sessions, positioning AI-assisted software engineering as a distinct premium product category.
