# Anthropic's Claude Code platform experienced a significant security incident when version 2.1.88 was inadvertently…

Anthropic's Claude Code platform experienced a significant security incident when version 2.1.88 was inadvertently published to npm without a proper .npmignore file, exposing 512,000 lines of TypeScript source code including 44 internal feature flags and unreleased agent capabilities. Among the exposed features are details of the "Kairos" roadmap update, which includes background agent execution and a "dream mode" for consolidating persistent AI memories. Coming five days after a separate Mythos CMS leak, security analysts have flagged this as a pattern raising questions about Anthropic's operational security ahead of its reported October 2026 IPO. A community Python reimplementation of Claude Code reached 50,000 GitHub stars within hours of the leak.
