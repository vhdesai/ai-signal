# Anthropic is in damage-control mode after source code for its Claude AI agent app was leaked, with reports that the…

Anthropic is in damage-control mode after source code for its Claude AI agent app was leaked, with reports that the signing system was cracked within 24 hours. Anthropic took down thousands of GitHub repos in an attempt to contain the spread — a move the company described as an accident. The extent of the breach (and whether model weights were exposed) remains undisclosed. A concurrent cyberattack on Mercor's LiteLLM open-source project heightened fears about AI infrastructure security broadly, with enterprise teams reassessing their open-source AI toolchain dependencies.
