# Google Gemini 3.1 Ultra Ships With 2M-Token Native Multimodal Context

Google's Gemini 3.1 Ultra is the marquee infrastructure release of the month, with a 2 million-token context window that operates natively across text, image, audio, and video without transcription intermediaries. A sandboxed Code Execution tool ships alongside it, letting the model write and run code mid-conversation. The release cements Google at the frontier and sets the stage for an anticipated Gemini 4 preview at I/O on May 19–20.
