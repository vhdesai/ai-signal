# Anthropic releases Claude Opus 4.8 with new "dynamic workflow" tool

Anthropic shipped Claude Opus 4.8 the same day it announced its $65B financing. The model touts gains in agentic task performance, advanced coding, and a stronger emphasis on honesty and self-correction. A new "dynamic workflow" capability lets the model plan and execute multi-step tasks more autonomously — extending the Claude Code franchise that now anchors the company's $47B revenue run rate.
