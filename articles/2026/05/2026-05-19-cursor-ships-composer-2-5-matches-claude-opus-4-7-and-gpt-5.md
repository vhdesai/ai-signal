# Cursor Ships Composer 2.5 — Matches Claude Opus 4.7 and GPT-5.5 at a Fraction of the Price

Cursor released Composer 2.5, built on Kimi K2.5 and trained on 25× more synthetic coding data than its predecessor, producing benchmark performance matching Claude Opus 4.7 and GPT-5.5 while significantly undercutting both on cost. CEO Michael Truell cited improvements in task persistence, multi-file instruction adherence, and reduced context drift on large codebases. Elon Musk confirmed the model was partially trained on SpaceXAI's Colossus 2 supercomputer — the same infrastructure Anthropic secured Colossus 1 for Claude Code. Cursor is doubling Composer 2.5 usage limits at no extra charge for the next week. 🔬
