# Microsoft Agent 365 Launches as Dedicated Enterprise AI Agent Control Plane Trending

Microsoft launched Agent 365 on May 1 as a dedicated orchestration and governance platform for enterprise AI agents within the Microsoft 365 ecosystem. The platform — part of Copilot Wave 3 — serves as a unified control plane for deploying, monitoring, and governing fleets of AI agents. It notably supports Claude, GPT, and Microsoft's own models in the same workflow, signaling Microsoft's multi-model strategy. This launch represents the clearest signal yet that enterprise AI has transitioned from individual copilots to coordinated, multi-agent systems.
