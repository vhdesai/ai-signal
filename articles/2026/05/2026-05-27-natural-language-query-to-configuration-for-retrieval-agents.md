# Natural-language query to configuration for retrieval agents (Zaharia et al.)

The paper proposes translating natural-language user requests into the configuration parameters retrieval agents need — chunking, embedding choice, retriever topology, system-and-control hooks. The framing crosses cs.AI and eess.SY, positioning RAG configuration as a control problem rather than a pure prompting one. The author list (including Zaharia) suggests this is part of the broader compound-AI-systems agenda from the Stanford / Berkeley / Databricks-affiliated labs.
