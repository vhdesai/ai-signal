# OpenAI's GPT-5.5 Instant became the default ChatGPT model on May 5, featuring transparent memory recall and faster…

OpenAI's GPT-5.5 Instant became the default ChatGPT model on May 5, featuring transparent memory recall and faster response times. Google shipped Gemini 3.1 Flash Lite (May 7–8) optimized for gateway and edge deployments. xAI's Grok 4.3 went live on the xAI API and X platform (April 30). No model has yet broken the Intelligence Index ceiling of 60.24 set by GPT-5.5 in April — the industry is currently catching its breath after a sprint of frontier releases.
