# Anthropic rolled out Claude Security in May 2026, a new capability that scans codebases for vulnerabilities, validates…

Anthropic rolled out Claude Security in May 2026, a new capability that scans codebases for vulnerabilities, validates findings to reduce false positives, and surfaces suggested patches for human review and approval before any changes land. The tool integrates directly with Claude Code's existing Routines and Managed Agents infrastructure, enabling it to run on scheduled scans or in response to new commits. It is the first direct product response from Anthropic to the security-focused positioning of GPT-5.5-Cyber and reflects the company's view that defensive tooling, not offensive capability, should be the primary enterprise offer for security workflows.
