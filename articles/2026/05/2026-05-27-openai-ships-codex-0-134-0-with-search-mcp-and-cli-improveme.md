# OpenAI ships Codex 0.134.0 with search, MCP, and CLI improvements

The release introduces case-insensitive local conversation-history search, per-server MCP environment targeting with OAuth options for streamable HTTP servers, and concurrent execution of read-only MCP tools. The --profile flag is now the primary selector across CLI, TUI, and sandbox flows. Windows TUI rendering corruption and websocket reliability also fixed. Research Breakthroughs
