# xAI's Grok Build Agent CLI Reviewed Following Beta Rollout

xAI's terminal-based agent CLI Grok Build entered fuller review coverage on May 26, ten days after a May 14 beta launch and the May 19 release of grok-build-0.1, an early-access coding model. Grok Build runs as an interactive TUI or headlessly in scripts and is compatible with the Agent Client Protocol — positioning xAI directly against Claude Code, Codex Cloud, and Cursor's Composer in the agentic-coding tooling race. 4.
