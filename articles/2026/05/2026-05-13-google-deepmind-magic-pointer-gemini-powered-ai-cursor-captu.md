# Google DeepMind "Magic Pointer": Gemini-Powered AI Cursor Captures Visual and Semantic Context

DeepMind published four interaction principles and live demos in Google AI Studio for an "AI-enabled pointer" that uses Gemini to understand what a cursor is pointing at — text, table cells, image regions — pairing that context with deictic voice commands like "fix this" or "what does this mean?" Integration into Chrome is planned, with deeper Magic Pointer features announced for the Googlebook laptop line. DeepMind CEO Demis Hassabis described early prototypes as "pretty magical."
