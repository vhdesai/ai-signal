# Trending Sakana AI: 7B Model Learns to Route Tasks Across GPT-5, Claude & Gemini via RL

Sakana AI published research demonstrating a compact 7B-parameter "orchestrator" model that dynamically routes tasks across GPT-5, Claude Sonnet 4, and Gemini 2.5 Pro — using reinforcement learning rather than static routing rules. The system selects the optimal frontier model for each sub-task in real time, offering a vendor-agnostic meta-layer for enterprise deployments. The approach has direct implications for cost efficiency: organizations can run premium models only for tasks that warrant them, using cheaper models for the rest, without manual workflow configuration.
