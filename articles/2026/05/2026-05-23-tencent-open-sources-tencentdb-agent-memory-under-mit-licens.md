# Tencent open-sources TencentDB Agent Memory under MIT license

Tencent released a fully local memory system for AI agents combining symbolic short-term memory with a four-tier long-term memory pyramid (Conversation → Atom → Scenario → Persona). Internal benchmarks report a 61.38% token reduction and 51.52% relative pass-rate gain on WideSearch, plus PersonaMem accuracy rising from 48% to 76% — a meaningful contribution to the agent-state-management problem enterprises are starting to encounter at scale.
