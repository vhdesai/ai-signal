# Cursor 3.0 Launches "Agents Window" for Parallel Agentic Code Execution Trending

Cursor 3.0 introduces an "Agents Window" that runs multiple parallel AI agents simultaneously to handle complex, multi-file development tasks — a platform-level redesign rather than an incremental feature update. Developers can now parallelize code generation, testing, and refactoring across independent agents in a single session. The move directly challenges GitHub Copilot's market position and traditional IDE tooling by offering true parallel agentic workflows natively in the development environment, a capability previously requiring bespoke agent orchestration frameworks.
