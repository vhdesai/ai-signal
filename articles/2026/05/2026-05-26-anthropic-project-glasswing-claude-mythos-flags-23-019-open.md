# Anthropic Project Glasswing: Claude Mythos flags 23,019 open-source vulnerabilities — patching is the new bottleneck

Claude Mythos Preview flagged 23,019 potential open-source vulnerabilities, with 6,202 estimated as high/critical severity. Of 1,752 findings reviewed by outside security firms, 90.6% were judged valid true positives. Anthropic has disclosed 530 high/critical bugs to maintainers but only 75 have been patched — "the volume of AI-found flaws is turning verification, disclosure, and patching into the new bottleneck." One example: a wolfSSL flaw allowing certificate forgery on a library used in billions of devices.
