# 🔥 HOT Google Gemini 3.1 Ultra: 2M-Token Native Multimodal Flagship

Google's Gemini 3.1 Ultra is the headline infrastructure release of the month, featuring a 2-million token context window that operates natively across text, image, audio, and video without transcription intermediaries. A sandboxed Code Execution tool ships alongside it, allowing the model to write and run code mid-conversation. The release cements Google's position at the frontier and sets the stage for what Demis Hassabis has called "Gemini 4 year" — expected to preview at Google I/O on May 19–20.
