# GitHub Copilot’s usage-based billing tests developer trust

TechCrunch reported developer backlash to GitHub Copilot’s shift from flat subscription pricing toward token-based usage billing. The strategic issue is not whether AI coding assistants are valuable, but whether enterprises can make consumption predictable enough for scaled adoption. For CIOs and engineering leaders, AI developer tooling is becoming a FinOps problem as much as a productivity one.
