# Inside OpenAI's New Founder Experience Team Led by Laura Modiano

OpenAI formalized a dedicated Founder Experience team under Laura Modiano (ex-Sequoia, ex-OpenAI Startup Fund), targeting seed and Series-A AI-native startups. The structure mirrors Stripe's Atlas program and is designed to lock in API choice at company-formation moment — a direct shot at AWS Activate and Microsoft for Startups. Worth a competitive briefing for the M12 / Founders Hub teams. C
