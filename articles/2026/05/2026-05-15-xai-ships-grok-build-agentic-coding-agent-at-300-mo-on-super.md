# xAI Ships Grok Build — Agentic Coding Agent at $300/mo on SuperGrok Heavy

xAI launched Grok Build in beta — its first dedicated agentic coding product — available to SuperGrok Heavy subscribers at $300 per month. Grok Build can autonomously write, debug, run, and iterate on multi-file codebases with minimal human prompting, positioning it as a direct challenger to Cursor, GitHub Copilot Workspace, and Replit Agent. The product leverages Grok 3.5's extended reasoning capabilities and xAI's in-house GPU cluster. Early testers report strong performance on Python and TypeScript full-stack tasks; broader language support is on the roadmap. 🆕
