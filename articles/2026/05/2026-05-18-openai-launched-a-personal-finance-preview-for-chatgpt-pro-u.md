# OpenAI launched a personal finance preview for ChatGPT Pro users in the U.S., enabling secure bank account connections…

OpenAI launched a personal finance preview for ChatGPT Pro users in the U.S., enabling secure bank account connections via Plaid (supporting 12,000+ financial institutions). Users get a dashboard covering portfolio performance, spending, subscriptions, upcoming payments, and savings goals. ChatGPT uses GPT-5.5's improved reasoning to answer financial planning questions grounded in the user's actual financial context. OpenAI explicitly notes the tool is not a replacement for professional financial advice. Rollout to Plus and other tiers is planned in coming weeks.
