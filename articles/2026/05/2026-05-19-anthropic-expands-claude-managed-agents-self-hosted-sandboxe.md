# Anthropic Expands Claude Managed Agents — Self-Hosted Sandboxes and MCP Tunnels Now Available

Anthropic is expanding Claude Managed Agents with two new infrastructure capabilities: self-hosted sandboxes (allowing companies to move their agents' tool execution into their own infrastructure) and MCP tunnels (enabling secure connections between Claude and internal systems). The update gives enterprises significantly more control over data residency and security posture without sacrificing the benefits of Anthropic's managed cloud. This follows the May 12 Code w/ Claude SF conference, where Anthropic announced doubled Claude Code rate limits, multiagent orchestration, "dreaming" memory refinement, and outcome-based evaluation webhooks.
