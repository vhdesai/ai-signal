# Anthropic releases Knowledge Work Plugins repo and Claude Code Plugins Directory

Two open-source releases landed on Anthropic's GitHub: a "knowledge-work-plugins" repo for Claude Cowork that specializes Claude for specific roles and teams, plus an officially curated directory of high-quality Claude Code plugins. Together they signal a more formal ecosystem strategy for extending Claude into role-specific knowledge work.
