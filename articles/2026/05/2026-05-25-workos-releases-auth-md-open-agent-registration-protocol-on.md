# WorkOS releases auth.md — open agent-registration protocol on OAuth standards

Most web applications still have no structured way for an AI agent to register. WorkOS's auth.md proposes a Markdown manifest apps publish at their domain telling agents which registration flows are supported, which scopes to request, and how to get user-linked credentials — without a human filling out a form. Foundational plumbing for the agentic web.
