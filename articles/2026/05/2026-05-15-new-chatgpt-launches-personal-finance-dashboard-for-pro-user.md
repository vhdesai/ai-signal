# 🟢 NEW ChatGPT Launches Personal Finance Dashboard for Pro Users via Plaid

OpenAI's ChatGPT can now connect to financial accounts through Plaid, giving Pro subscribers a personal finance dashboard with read-only access to balances, transactions, investments, subscriptions, upcoming bills, and savings goals. The feature marks OpenAI's entry into the personal financial intelligence market, enabling conversational analysis of spending patterns and financial planning directly within ChatGPT. Access is currently limited to ChatGPT Pro users at $200/month.
