# Raindrop Releases "Workshop" — Open-Source Local AI Agent Debugger

Raindrop has open-sourced "Workshop," a local-first debugging and evaluation framework for AI agents that runs entirely on-device without requiring cloud API calls. Workshop provides step-through debugging for multi-step agentic pipelines, allowing developers to inspect intermediate reasoning states, tool call results, and memory states at each decision point. The tool supports LangGraph, CrewAI, and custom agent frameworks via an adapter interface. At a time when most agent observability tools are SaaS-only, Workshop's local execution model is a meaningful differentiator for enterprise teams with strict data governance requirements.
