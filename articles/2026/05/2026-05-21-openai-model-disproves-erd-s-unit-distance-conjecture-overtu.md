# OpenAI Model Disproves Erdős Unit-Distance Conjecture, Overturning 80-Year Assumption

OpenAI's internal reasoning model produced a counterexample to Paul Erdős's 1946 unit-distance conjecture, autonomously generating a construction that mathematicians have spent eight decades unable to refute. The result is being independently checked, but if it holds, it marks the first time an AI model has overturned a foundational open problem in discrete geometry — a milestone for AI-assisted mathematical discovery.
