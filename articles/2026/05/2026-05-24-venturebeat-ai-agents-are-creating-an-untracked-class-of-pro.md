# VentureBeat: AI Agents Are Creating an Untracked Class of Production Failures

A new VentureBeat analysis flags an emerging category of incidents enterprises aren't tracking: agent-initiated actions that are technically correct given incomplete context, but cascade through downstream infrastructure. With 79% of organizations now running agents in production and Gartner projecting 33% of enterprise software will be agentic by 2028, the lack of a unified postmortem framework is becoming a measurable risk.
