# Anthropic's Claude Mythos Moves Toward Public Release via Claude Code and Claude Security

Anthropic's restricted "Mythos" capability — previously limited to red-team and vetted enterprise partners — is being staged into the commercial Claude Code product line and the new Claude Security dashboard. The release shifts Mythos from a research preview into a board-visible governance tool, marking the first time a frontier lab has productized an alignment-focused sub-model. Enterprise customers should expect parallel rollout into Claude.ai and AWS Bedrock channels.
