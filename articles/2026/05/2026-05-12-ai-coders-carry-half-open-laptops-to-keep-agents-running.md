# AI Coders Carry Half-Open Laptops to Keep Agents Running

As long-running AI coding agents become production tools, developers are physically leaving their laptops ajar — through airports, offices, even ice rinks — to keep sessions alive. The cultural artifact mirrors a real shift: agent runtime length is starting to dictate user behavior.
