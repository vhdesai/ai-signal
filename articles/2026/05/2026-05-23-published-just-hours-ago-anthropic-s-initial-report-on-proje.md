# Published just hours ago, Anthropic's initial report on Project Glasswing reveals that its unreleased Claude Mythos…

Published just hours ago, Anthropic's initial report on Project Glasswing reveals that its unreleased Claude Mythos Preview model found over 10,000 vulnerabilities across ~50 partner organizations in just one month. Partners report bug-finding rates increased by more than 10×. Key disclosures: Cloudflare found 2,000 bugs (400 high or critical); Mozilla found and fixed 271 Firefox vulnerabilities — 10× more than via older models. This milestone is significant for Microsoft's Corp Dev team given the model's stated ability to find bugs faster than developers can patch them, raising supply/demand questions in AI-driven security tooling. Source: Engadget, The Decoder (May 23, 2026)
