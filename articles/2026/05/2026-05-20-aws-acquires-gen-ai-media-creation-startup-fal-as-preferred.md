# AWS Acquires Gen-AI Media Creation Startup fal as Preferred Cloud Provider

Amazon Web Services confirmed on May 20 that it has acquired fal, a fast-growing generative AI media creation startup, naming it its preferred cloud provider for large media conglomerates. The deal gives AWS a managed service play for state-of-the-art AI video and image tools inside a secure, IP-protected enterprise environment. The move signals AWS is actively competing with Google and Azure for the booming media-AI vertical.
