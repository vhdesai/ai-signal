# Amazon's Trainium Starts Winning Over AI Developers as Nvidia Alternative

Amazon's long-running effort to build a credible Nvidia alternative is gaining traction. Anthropic and OpenAI have already committed to renting large amounts of current and future Trainium capacity, and recent software improvements are now pulling smaller developers in as well. Documentation and tooling — historically Amazon's weak point — have improved markedly, narrowing the gap with the CUDA ecosystem.
