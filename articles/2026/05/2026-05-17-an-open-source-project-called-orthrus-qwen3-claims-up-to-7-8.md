# An open-source project called Orthrus-Qwen3 claims up to 7.8x tokens-per-forward-pass speedup on Qwen3 models,…

An open-source project called Orthrus-Qwen3 claims up to 7.8x tokens-per-forward-pass speedup on Qwen3 models, reportedly with identical output distributions. The project garnered 155 HN points and 24 comments and is being watched closely by teams running Qwen models in production. Independent verification is still pending, but the technique has drawn early interest from inference optimization engineers.
