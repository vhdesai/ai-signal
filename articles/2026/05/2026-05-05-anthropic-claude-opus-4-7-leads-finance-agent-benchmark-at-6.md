# Anthropic Claude Opus 4.7 — Leads Finance Agent Benchmark at 64.37%, Beats GPT-5.5

Claude Opus 4.7 powers Anthropic's 10 new financial services AI agents, launched at an invite-only New York event with JPMorgan CEO Jamie Dimon. On Vals AI's Finance Agent benchmark, it scores 64.37% — ahead of GPT-5.5 (59.96%) and Gemini 3.1 Pro (59.72%). The agents include pitch builder, earnings reviewer, GL reconciler, and KYC screener. FIS is already using agents for AML case reviews; Carlyle has deployed Claude across investment analysis and portfolio oversight. Claude now also works as a single agent across the full Microsoft 365 suite (Excel, PowerPoint, Word, Outlook).
