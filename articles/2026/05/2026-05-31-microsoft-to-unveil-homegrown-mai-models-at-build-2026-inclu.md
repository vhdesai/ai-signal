# Microsoft to Unveil Homegrown MAI Models at Build 2026, Including a Coding Model

Reuters and The Information reported that Microsoft will debut its in-house MAI model family at Build 2026, opening June 2, including a coding model explicitly aimed at winning back GitHub Copilot share from Claude Code, which has overtaken Copilot as the dominant developer AI tool. The move signals Microsoft pushing toward greater model independence alongside its OpenAI partnership.
