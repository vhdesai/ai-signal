# Anthropic held its "Code w/ Claude 2026" event, with technologist Simon Willison live-blogging morning keynote sessions…

Anthropic held its "Code w/ Claude 2026" event, with technologist Simon Willison live-blogging morning keynote sessions that covered new developments in Claude Code and multi-step agentic workflows. The event reinforced Anthropic's push to position Claude as the preferred coding assistant for senior engineers and enterprises — not just consumer chatbot users. Willison's live coverage noted a convergence theme: the line between "vibe coding" and "agentic engineering" is blurring as scaffolding becomes more sophisticated. No specific model releases were confirmed, but new Claude Code capabilities were showcased.
