# xAI Launches Grok Build — Agentic Coding Agent Targeting Professional Developers

Elon Musk's xAI launched Grok Build, its first dedicated AI coding agent, available to SuperGrok Heavy subscribers at $300/month. The tool offers a "plan mode," CLI integration, and a new partnership with Cursor to rebuild coding capabilities from the ground up. Musk explicitly acknowledged xAI has lagged behind Anthropic and OpenAI in professional coding; Grok Build is the company's direct response to close that gap. The launch follows the SpaceX-xAI merger and significant compute expansion.
