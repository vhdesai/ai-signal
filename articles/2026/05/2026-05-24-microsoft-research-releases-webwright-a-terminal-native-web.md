# Microsoft Research releases Webwright — a terminal-native web-agent framework

Microsoft Research released Webwright, a terminal-native web-agent framework that scores 60.1% on the Odysseys benchmark — up from base GPT-5.4's 33.5%. The framework wraps a frontier model in a shell-native harness, suggesting MSR is investing in agent scaffolding as a primary lever for capability gains rather than waiting on new base models.
