# Microsoft Releases Fara1.5 Browser Agents (4B / 9B / 27B) — 27B Hits 72% on Online-Mind2Web Benchmark

Microsoft released Fara1.5, a family of three browser-use computer-agent models at 4B, 9B, and 27B parameters. The 27B variant achieves 72% on the Online-Mind2Web benchmark for autonomous web navigation, enabling complex multi-step tasks like form filling, web research, and e-commerce workflows without human intervention. The models are designed for enterprise process automation and extend Microsoft's agentic AI strategy beyond Copilot into fully autonomous desktop and browser operation.
