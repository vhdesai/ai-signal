# Cursor 3.4 Introduces Cloud Agent Dev Environments with Full Audit & Governance

Cursor's latest release enables teams to configure development environments for cloud agents — including multi-repo setups, Dockerfile-based environment-as-code with build secrets, and full version history with admin-only rollback controls. Admins now get an audit log for every agent environment action and the ability to scope secrets and egress per environment. The update follows the May 11 launch of @Cursor in Microsoft Teams, enabling agents to be delegated tasks directly from Teams channels.
