# Startup Subquadratic launched SubQ 1M-Preview with $29M in seed funding, claiming to be the first commercially…

Startup Subquadratic launched SubQ 1M-Preview with $29M in seed funding, claiming to be the first commercially available LLM built on sparse subquadratic attention — not a standard transformer. The model ships with a native 12 million token context window and claims ~1/5 the cost of frontier models on long-context tasks. Vendor-stated performance figures have yet to be independently confirmed, but the architecture represents a genuine departure from the status quo if benchmarks hold up.
