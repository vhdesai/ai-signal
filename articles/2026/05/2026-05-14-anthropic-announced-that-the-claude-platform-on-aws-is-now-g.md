# Anthropic announced that the Claude Platform on AWS is now generally available, offering full-feature parity with the…

Anthropic announced that the Claude Platform on AWS is now generally available, offering full-feature parity with the native Claude API while leveraging AWS IAM authentication, CloudTrail audit logging, and AWS billing and commitment retirement. Customers can deploy Claude Managed Agents at scale across most AWS commercial regions. The release is the enterprise counterpart to the Claude Agent SDK launch (May 6) and positions Anthropic's agentic stack as cloud-agnostic with both AWS and Azure integrations now live.
