# ### First Windows PCs Using Nvidia Chips as Main Processor Debut at Computex

Nvidia and Microsoft are set to introduce the first Windows PCs that use an Nvidia chip as the main processor, debuting next week at Computex with Surface and Dell among the launch devices. The shift puts Nvidia into the client CPU role long held by x86 incumbents and tightens the Microsoft–Nvidia stack from data center down to the desktop — a structural change to the Windows hardware supply chain.
