# xAI's Grok officially integrated into OpenClaw via OAuth

xAI made Grok 4.3 the default model option inside the NVIDIA-backed OpenClaw agentic OS, available to SuperGrok ($30/mo) and X Premium ($8/mo) subscribers via OAuth without a separate API key. Grok remains the only major LLM with native access to live X data, positioning it for social-media monitoring and real-time news agents. Research Breakthroughs G Breaking Research
