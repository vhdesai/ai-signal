# Microsoft Research releases Webwright — terminal-native web agent at 60.1% on Odysseys

Webwright is a terminal-native web-agent framework that scores 60.1% on the Odysseys benchmark, up from base GPT-5.4's 33.5%. The jump shows Microsoft's research arm is treating browser-control agents as a first-class research surface — relevant context for Copilot's enterprise roadmap.
