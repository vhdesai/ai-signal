# Gemini 3.1 Ultra Already Shipping with 2M-Token Native Multimodal Context

Google's Gemini 3.1 Ultra — the headline model of early May — operates natively across text, image, audio, and video with a 2-million token context window and no transcription intermediaries. A sandboxed Code Execution tool ships alongside it, allowing the model to write and run code mid-conversation. The release positions Gemini as a forcing function on context length across the frontier-model field.
