# OpenAI Discloses Security Incident: Code Repository Data Stolen in Targeted Attack

OpenAI disclosed a security incident in which attackers exfiltrated data from the company's internal code repositories, including portions of internal tooling and infrastructure code. OpenAI stated that model weights and customer data were not compromised, but acknowledged that the stolen code could provide adversaries with insights into OpenAI's system architecture and deployment practices. The company has notified affected parties and is cooperating with law enforcement. The disclosure comes amid heightened scrutiny of AI lab security practices, and follows a 2025 incident in which OpenAI's internal messaging systems were breached.
