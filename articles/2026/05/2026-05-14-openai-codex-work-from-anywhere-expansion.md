# OpenAI Codex: "Work From Anywhere" Expansion

OpenAI published a product update enabling developers to work with Codex from any device or environment, significantly expanding the reach of its agentic coding platform. This follows the April 23 GPT-5.5 launch and comes as OpenAI directly competes with Anthropic's Claude Code in the enterprise developer tooling market. The same day, OpenAI also updated ChatGPT's ability to better recognize context in sensitive conversations, improving guardrails for consumer deployments.
