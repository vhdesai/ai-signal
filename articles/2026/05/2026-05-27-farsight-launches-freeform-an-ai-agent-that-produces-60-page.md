# Farsight Launches Freeform — An AI Agent That Produces 60+ Page CIMs from a Single Prompt

Investment-banking-tooling vendor Farsight launched Freeform, an autonomous agent that drafts full Confidential Information Memoranda — typically 60-plus pages — from a single prompt, scaffolded by the firm's prior deal work. The product directly addresses the second-year-analyst grunt-work category and is positioned for mid-market M&A advisory shops. Notable for Corp Dev teams: this is the first agent product with sufficient document-length capability to threaten internal deal teams' baseline outputs.
