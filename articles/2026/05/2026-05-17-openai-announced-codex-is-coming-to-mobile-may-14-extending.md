# OpenAI announced Codex is coming to mobile (May 14), extending its agentic coding platform to phones

OpenAI announced Codex is coming to mobile (May 14), extending its agentic coding platform to phones. Amazon launched an AI shopping assistant for the search bar powered by Alexa+ (May 13). Notion converted its workspace into a hub for AI agents the same week. Databricks also announced it is integrating GPT-5.5 into enterprise agent workflows (May 15). The pattern is consistent: frontier models are moving from chat to action-oriented agent interfaces embedded in existing workflows.
