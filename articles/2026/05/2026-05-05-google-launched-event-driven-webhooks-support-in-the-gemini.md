# Google launched event-driven Webhooks support in the Gemini API on May 4, enabling developers to replace polling-based…

Google launched event-driven Webhooks support in the Gemini API on May 4, enabling developers to replace polling-based workflows with push-based notifications for Batch API and long-running operations — a significant developer experience improvement for agentic pipelines. Earlier this spring, Google also released the updated Gemini Robotics ER 1.6 preview (April 14) with improved instrument reading and spatial reasoning, and launched a new Gemini 3.1 Flash TTS Preview (April 15) optimized for cost-efficient, steerable text-to-speech.
