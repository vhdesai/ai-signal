# Anthropic adds enterprise controls for managed Claude agents

Anthropic shipped MCP tunnels and self-hosted sandboxes for Claude Managed Agents, addressing enterprise concerns around private-network access and execution environments. The capabilities are aimed at letting agents operate closer to sensitive internal systems without requiring broad internet exposure. The timing, alongside Google’s agent push, underscores how fast the enterprise agent stack is hardening around security, deployment and governance requirements.
