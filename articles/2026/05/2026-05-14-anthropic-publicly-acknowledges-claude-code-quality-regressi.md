# Anthropic Publicly Acknowledges Claude Code Quality Regression After User Reports

Anthropic acknowledged a measurable quality regression in Claude Code — its agentic software development tool — following a deployment update, after a wave of user reports described increased rates of incomplete edits, hallucinated file paths, and premature task termination. The company stated the regression was introduced in a recent infrastructure update and that a fix is being rolled out, with full resolution expected within 48 hours. The public acknowledgment is notable as AI coding tools have rarely admitted performance regressions with this level of specificity. The incident underscores the operational complexity of maintaining consistent quality in rapidly evolving agentic AI products that are deeply integrated into developer workflows. 🎓 Academic Research
