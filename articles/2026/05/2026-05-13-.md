# 📈

Google's Gemini 3.1 Ultra is the headline infrastructure release of May 2026, featuring a 2-million-token context window that operates natively across text, image, audio, and video without transcription intermediaries. A sandboxed Code Execution tool ships alongside it, letting the model write and run code mid-conversation. The release comes ahead of Google I/O 2026 (May 19–20) where further Gemini announcements are expected, and ahead of tomorrow's Google Android Show, where Gemini integration into Android 17 and Chrome AI upgrades is anticipated.
