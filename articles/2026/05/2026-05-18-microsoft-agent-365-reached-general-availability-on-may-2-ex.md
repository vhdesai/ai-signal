# Microsoft Agent 365 reached general availability on May 2, extending enterprise-grade identity, security, and…

Microsoft Agent 365 reached general availability on May 2, extending enterprise-grade identity, security, and governance tooling to AI agents across M365 environments — a key enabler for organizations deploying agentic workflows at scale. On May 6, Anthropic opened the Claude Agent SDK to all external developers (previously invite-only), alongside the launch of Claude Code Auto Mode, which expands autonomous coding agent capabilities. Anthropic also launched ten financial-services agents in partnership with JPMorgan in the same week.
