# Microsoft's license to OpenAI IP becomes non-exclusive through 2032; OpenAI products will still ship first on Azure but can run anywhere. Microsoft no longer pays OpenAI a revenue share, while OpenAI continues paying Microsoft through 2030. The deal clears the legal path for OpenAI's $50B AWS commitment for its Frontier agent service. OpenAI also converted to a public benefit corporation. The closed loop that defined enterprise AI from 2022–2025 is over.
