# Microsoft Fara1.5: Browser Computer-Use Agents Outperform OpenAI Operator & Gemini 2.5  Hot

Microsoft released Fara1.5, a family of browser computer-use agents in 4B, 9B, and 27B parameter sizes that outperform OpenAI Operator and Gemini 2.5 Computer Use on the Online-Mind2Web benchmark. Even the smallest 4B model crosses the Operator baseline, materially lowering the cost-to-deploy floor for browser automation. The release demonstrates that compact open weights can lead on practical agentic benchmarks and extends Microsoft's enterprise-agent push beyond Copilot. 🔬 Research Breakthroughs 7 items
