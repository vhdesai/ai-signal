# OpenBMB Releases MiniCPM-V 4.6 (1.3B) — Most Recent Model Ship as of Today New

OpenBMB released MiniCPM-V 4.6 with 1.3 billion parameters on May 11, the most recently tracked frontier model as of this digest. With a 262K-token context window and open-source availability, it targets on-device and embedded inference use cases where cloud API costs are prohibitive. The model continues the trend of capable, compact multimodal models closing the capability gap with much larger proprietary systems for narrow deployment scenarios.
