# NVIDIA releases Gated DeltaNet-2 — linear attention with decoupled erase and write

NVIDIA released Gated DeltaNet-2, a linear-attention layer that decouples the erase and write operations in the Delta Rule for more controllable memory editing in long-context models. The architecture is part of a broader 2026 push to reduce quadratic attention costs without giving up reasoning quality. S Research
