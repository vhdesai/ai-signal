# Anthropic Acquires Stainless, Shuts Off SDK Access for OpenAI and Google

Anthropic acquired Stainless — the developer tools startup that built the SDK infrastructure underlying OpenAI, Google, and Cloudflare's developer ecosystems — for a reported price north of $300 million, then shut down the hosted products for rivals. Founded by former Stripe engineer Alex Rattray and backed by Sequoia and a16z, Stainless had been deeply embedded across the AI ecosystem's tooling layer. The acquisition is widely interpreted as a strategic infrastructure play: by controlling the SDK generation layer, Anthropic can reduce the integration friction for Claude while raising it for competitors. This is an agentic AI infrastructure move, not a pure model-capability story.
