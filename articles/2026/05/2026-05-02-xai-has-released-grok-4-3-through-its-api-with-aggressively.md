# xAI has released Grok 4.3 through its API with aggressively competitive pricing targeting enterprise developers

xAI has released Grok 4.3 through its API with aggressively competitive pricing targeting enterprise developers. Alongside the model update, xAI unveiled a Custom Voices voice-cloning suite that allows developers to create personalized synthetic speech experiences. The release positions xAI directly against OpenAI's GPT-4o voice capabilities and ElevenLabs in the audio-AI market. Availability expanded rapidly through Vercel and other hosting partners within 24 hours of launch.
