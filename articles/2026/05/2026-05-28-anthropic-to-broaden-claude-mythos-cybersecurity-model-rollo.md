# Anthropic to broaden Claude Mythos cybersecurity model rollout

Alongside the funding announcement, Anthropic said it will widen access to Mythos — the cybersecurity-grade model scoring 97.6% on USAMO 2026 — which, via Project Glasswing, has already surfaced more than 10,000 previously unknown software vulnerabilities for partners including AWS, Apple, Microsoft, JPMorgan, NVIDIA, Palo Alto Networks, and the Linux Foundation. Mythos-class access will reach broader customers within weeks, raising new dual-use questions for regulators.
