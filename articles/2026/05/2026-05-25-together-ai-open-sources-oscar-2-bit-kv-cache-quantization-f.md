# Together AI open-sources OSCAR — 2-bit KV-cache quantization for long-context serving

OSCAR is an attention-aware 2-bit KV-cache quantization system designed to slash memory footprint for long-context LLM serving without measurable quality loss on standard benchmarks. Released as open source, it targets inference-cost-sensitive deployments where 1M-token contexts have become table stakes.
