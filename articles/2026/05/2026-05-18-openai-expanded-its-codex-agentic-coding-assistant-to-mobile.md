# OpenAI expanded its Codex agentic coding assistant to mobile platforms (May 15), enabling on-the-go code generation and…

OpenAI expanded its Codex agentic coding assistant to mobile platforms (May 15), enabling on-the-go code generation and review for developers. Separately, Anthropic's Claude Mythos has appeared in Google Cloud's model catalog without the usual "Preview" label — an unusual status that analysts interpret as indicating enterprise-readiness despite the absence of a formal public launch. The absence of the preview tag in Google Cloud's interface is the strongest signal yet that a broader Mythos rollout may be imminent under Anthropic's $200B Google Cloud contract.
