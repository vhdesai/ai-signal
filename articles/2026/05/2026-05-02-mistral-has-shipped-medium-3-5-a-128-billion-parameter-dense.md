# Mistral has shipped Medium 3.5, a 128-billion-parameter dense merged model released under open weights

Mistral has shipped Medium 3.5, a 128-billion-parameter dense merged model released under open weights. The model consolidates chat, multi-step reasoning, and code generation into a single architecture, challenging proprietary offerings in the mid-tier frontier segment. Mistral is positioning Medium 3.5 as a practical enterprise choice for organizations that want frontier-grade capability with on-premise deployment flexibility. Initial benchmarks show strong performance on MMLU, HumanEval, and MATH relative to model size.
