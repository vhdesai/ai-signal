# GPT-5.5 is OpenAI's most capable and first ground-up retrained model since GPT-4.5 — every 5.1–5.4 release was a…

GPT-5.5 is OpenAI's most capable and first ground-up retrained model since GPT-4.5 — every 5.1–5.4 release was a post-training iteration on the same base. With a 1M-token context window and a new agent-oriented architecture, it scores 82.7% on Terminal-Bench 2.0 (vs. 75.1% for GPT-5.4, 69.4% for Claude Opus 4.7), and 84.9% on GDPval, OpenAI's knowledge-work benchmark. The model is priced at $5/$30 per 1M tokens — a 2× increase over GPT-5.4 — but uses ~40% fewer tokens per task. One caveat: third-party evaluators note GPT-5.5 leads on planning-and-execution benchmarks but trails Claude Opus 4.7 on codebase resolution (SWE-Bench Pro: 58.6% vs. 64.3%) and multilingual tasks.
