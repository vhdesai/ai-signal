# Cursor 3.0 — Parallel Agent Coding IDE

Cursor 3.0 launched an "Agents Window" as a central workspace for orchestrating multiple AI coding agents simultaneously — running locally, in git worktrees, in the cloud, or over SSH. Developers can now spin up parallel agent workflows executing independent tasks concurrently, dramatically compressing build and review cycles. The release aligns with Cursor's separately announced TypeScript SDK for agent runtime. (Source: MSN)
