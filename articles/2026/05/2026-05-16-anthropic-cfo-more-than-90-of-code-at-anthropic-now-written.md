# Anthropic CFO: More Than 90% of Code at Anthropic Now Written by Claude Code

Anthropic CFO Krishna Rao disclosed today that over 90% of the company's internal codebase is now produced by Claude Code, the company's AI-native coding agent. Rao described the shift as a "step-change in engineering productivity," with human engineers increasingly in a supervisory and architectural role rather than writing code line by line. The disclosure is the most concrete data point yet from a frontier AI lab on the real-world penetration of AI coding tools at scale. It underscores Anthropic's dual position as both the maker and a major customer of its own AI technology. 🔥 HOT 📈
