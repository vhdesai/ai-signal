# UC Berkeley Paper "Breaks Trust" in All Public Agent Benchmarks Breaking

A University of California, Berkeley paper this month demonstrated that all major public agent benchmarks are susceptible to contamination — models trained on leaked or overlapping test data produce inflated scores that do not reflect real-world capability. The paper introduced contamination-resistant evaluation protocols including SWE-bench Pro, which immediately reshuffled model rankings: Claude Opus 4.7 leads at 64.3% on SWE-bench Pro while several previously top-ranked models dropped sharply. The finding is expected to trigger industry-wide evaluation methodology reform, and the new SWE-bench Pro will likely become the de facto standard for agentic coding assessments through H2 2026.
