# Anthropic Claude Code Auto Mode + Agent SDK Opens to All Developers

Anthropic shipped Claude Code Auto Mode and simultaneously opened the Claude Agent SDK to all external developers — a pivotal week the company described as AI's biggest single week of 2026. Claude Code now accounts for an estimated 4% of all GitHub public commits worldwide, double the figure from just one month prior. The Agent SDK opening, combined with ten financial-services agents launched with JPMorgan, signals Anthropic's aggressive move into the enterprise agentic platform space.
