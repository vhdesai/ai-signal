# Google DeepMind is experimenting with treating cursor position as a real-time signal for Gemini, allowing the model to…

Google DeepMind is experimenting with treating cursor position as a real-time signal for Gemini, allowing the model to infer user intent from what a cursor hovers over rather than requiring explicit text prompts. The prototype — called the "AI Pointer" — lets users hover over a data table, say "make a pie chart," and receive an instant response without copying content into a chat window. The research aims to eliminate the friction of writing precise instructions for LLM-based assistants by embedding context directly in UI interaction. If shipped, it would mark a fundamental shift in how AI assistants are accessed on desktop.
