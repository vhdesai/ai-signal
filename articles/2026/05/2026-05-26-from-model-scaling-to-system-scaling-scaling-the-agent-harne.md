# From Model Scaling to System Scaling: scaling the agent "harness"

Argues — with empirical scaling curves — that the next frontier gains will come from scaling the surrounding harness (tools, memory, orchestration, verifiers) rather than model parameters alone. Proposes an explicit alternative scaling law for agent systems and a way to measure harness compute. Gives CTOs evidence to redirect AI budget from model training toward agent infrastructure.
