# SubQ Launches First Commercial Subquadratic LLM with 12M-Token Context

Startup Subquadratic launched SubQ 1M-Preview with $29M seed funding, claiming the first commercially available LLM built on sparse subquadratic attention — not a standard transformer. The model ships with a native 12 million token context window and claims roughly one-fifth the cost of frontier models on long-context tasks. SubQ Code, a repo-wide coding agent, accompanies the API launch. Independent benchmarks are still pending.
