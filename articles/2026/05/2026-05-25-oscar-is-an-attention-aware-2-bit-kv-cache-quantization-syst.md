# OSCAR is an attention-aware 2-bit KV-cache quantization system designed to make long-context inference dramatically cheaper.

OSCAR is an attention-aware 2-bit KV-cache quantization system designed to make long-context inference dramatically cheaper. The release matters for any team serving models above ~200K tokens, where KV-cache memory has become the dominant inference cost driver. The open-source posture is also a strategic move to commoditize a layer where hyperscalers currently extract premium pricing.
