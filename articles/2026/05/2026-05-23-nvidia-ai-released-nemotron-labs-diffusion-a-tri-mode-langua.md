# NVIDIA AI released Nemotron-Labs-Diffusion, a tri-mode language model achieving 6× more tokens per forward pass…

NVIDIA AI released Nemotron-Labs-Diffusion, a tri-mode language model achieving 6× more tokens per forward pass compared to Qwen3-8B. The release targets efficient inference at scale and represents NVIDIA's growing push to participate in the model layer, not just the chip layer.
