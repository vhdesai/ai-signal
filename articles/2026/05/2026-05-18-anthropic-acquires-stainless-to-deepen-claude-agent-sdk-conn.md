# Anthropic Acquires Stainless to Deepen Claude Agent SDK Connectivity

Anthropic acquired Stainless, a developer tooling company specializing in API SDK generation and integration infrastructure, to accelerate Claude's connectivity with external developer tools and enterprise systems. The acquisition expands Claude's agent ecosystem at a moment when Claude Code SDK adoption is driving the majority of Anthropic's high-margin API revenue growth — and positions the company to deepen integration with the enterprise toolchains where OpenAI's Codex is also competing. 💼 Industry News & Deals
