# Hot xAI Launches Grok Build — Coding Agent for Developers at $300/Month

Elon Musk's xAI released Grok Build in early beta — a command-line coding agent for SuperGrok Heavy subscribers at $300/month. Developers aim Grok Build at a codebase and describe a task in natural language; the agent inspects the project, plans the changes, and executes them. The launch puts xAI in direct competition with Claude Code, OpenAI Codex, and Cursor in the fast-growing AI-native developer workflow market. A full Grok 4.3 model release was noted as recent on llm-stats.com.
