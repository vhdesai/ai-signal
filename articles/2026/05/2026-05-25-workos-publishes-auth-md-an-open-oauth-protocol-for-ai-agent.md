# WorkOS publishes auth.md — an open OAuth protocol for AI agents as first-class enterprise clients

WorkOS opened a proposed standard for registering AI agents as OAuth clients with enterprise identity providers. The protocol addresses one of the most-cited blockers to enterprise agent rollout: scoping, auditing, and revoking agent access at the IdP layer rather than the application layer.
