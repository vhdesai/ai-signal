# Google Gemini 3.1 Ultra — 2M Token Context, Native Multimodal, Built-In Code Execution

Google's Gemini 3.1 Ultra is the headline model release of May. Its 2-million token context window operates natively across text, image, audio, and video without transcription intermediaries. A sandboxed Code Execution tool ships alongside it, letting the model write and run code mid-conversation. Earlier Gemini Deep Think variants earned a gold medal at the International Mathematical Olympiad, per Stanford's 2026 AI Index.
