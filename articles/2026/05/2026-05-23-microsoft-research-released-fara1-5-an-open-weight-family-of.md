# Microsoft Research released Fara1.5, an open-weight family of browser computer-use agents in 4B, 9B, and 27B parameter…

Microsoft Research released Fara1.5, an open-weight family of browser computer-use agents in 4B, 9B, and 27B parameter sizes, built on fine-tuned Qwen 3.5. The flagship Fara1.5-27B scored 72% on Online-Mind2Web — the industry's toughest live-web benchmark — surpassing OpenAI Operator (58.3%) and Gemini 2.5 Computer Use (57.3%). The 9B model is immediately available on Azure AI Foundry; the 4B and 27B variants arrive shortly. The release also includes FaraGen1.5, a synthetic data pipeline for training agents on real-world web tasks. Source: MarkTechPost, Decrypt (May 22, 2026)
