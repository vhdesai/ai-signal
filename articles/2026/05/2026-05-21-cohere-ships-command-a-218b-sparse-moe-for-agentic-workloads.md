# Cohere ships Command A+: 218B Sparse MoE for agentic workloads

Cohere consolidated four prior Command A variants into a single 218B Sparse Mixture-of-Experts model, runnable on just two H100 GPUs at W4A4 quantization. It supports 48 languages and is Cohere's first multimodal reasoning model — a notable signal that mid-size labs are finding capital-efficient paths to frontier-adjacent capability through MoE consolidation.
