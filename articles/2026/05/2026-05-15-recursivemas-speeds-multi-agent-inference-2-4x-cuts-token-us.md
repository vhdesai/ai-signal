# RecursiveMAS Speeds Multi-Agent Inference 2.4x, Cuts Token Usage 75%

Researchers from UIUC and Stanford published RecursiveMAS, a multi-agent framework that lets AI agents share embeddings instead of raw text when communicating — slashing token usage by 75% and cutting training costs by more than half while achieving 2.4x inference throughput gains. VentureBeat highlighted the practical enterprise implication: teams running large agent pipelines can dramatically reduce both latency and API cost without sacrificing task quality. 3 · Products & Tools
