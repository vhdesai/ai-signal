# Microsoft ships Excel Copilot redesign with live web-data formulas

Microsoft consolidated Excel's Copilot UX down to two entry points and added a dockable Copilot panel, refreshed keyboard shortcuts, and curated smart suggestions for Windows and Mac. The bigger functional change: Excel formulas can now pull live web data directly through Copilot, plus new AI transparency tooling lets users audit where a Copilot-generated answer came from. It is Microsoft's clearest move yet toward Copilot as a built-in workspace layer rather than a side panel.
