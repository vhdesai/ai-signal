# Cursor Composer 2.5 Officially Launches: Matching Opus 4.7 & GPT-5.5 at 1/10th the Cost  Hot

Cursor's in-house coding model Composer 2.5 — built on Moonshot's Kimi K2.5 checkpoint with 25× more synthetic tasks and a targeted RL technique — reaches SWE-Bench Multilingual 79.8% and CursorBench v3.1 63.2%, matching Claude Opus 4.7 and GPT-5.5 at roughly one-tenth the cost ($0.50/M input tokens). Cursor also disclosed it is training a larger successor on SpaceX/xAI's Colossus 2 supercomputer using 10× more compute. The launch signals that price-performance, not raw capability, is now the primary battleground in AI coding tools.
