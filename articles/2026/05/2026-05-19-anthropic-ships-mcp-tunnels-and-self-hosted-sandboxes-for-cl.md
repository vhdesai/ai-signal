# Anthropic ships MCP tunnels and self-hosted sandboxes for Claude Managed Agents

Anthropic shipped two enterprise-grade features for Claude Managed Agents: MCP tunnels (private network routing to internal MCP servers without internet exposure) and self-hosted sandboxes (agent execution environments running inside the customer's own infrastructure under their security controls). The features directly address the top enterprise objections to running agents on third-party clouds, and land on the same day as Google's Gemini Spark launch — underscoring the accelerating race for enterprise-agent market share.
