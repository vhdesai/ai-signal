# xAI's Grok V9-Medium finishes training at 1.5T parameters, targets mid-June release

Elon Musk announced that xAI's V9-Medium — three times larger than the model currently serving Grok production traffic — has completed pretraining with positive evaluation results. Supervised fine-tuning is underway and reinforcement learning begins this week. The model was trained on Cursor developer data to target coding leadership, and xAI committed to open-source the current 0.5T model by year-end.
