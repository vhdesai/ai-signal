# Cognition launches Devin Security Swarm for autonomous vulnerability remediation

Cognition, maker of the Devin coding agent, launched Devin Security Swarm, which uses a new "Agentic MapReduce" architecture — parallel agents that reason across a codebase, reproduce exploits in sandboxes, and open remediation pull requests. On a 50-vulnerability benchmark tied to real GitHub Security Advisories across 14 languages, it reported 72% recall (36/50), ahead of the other tools tested, at roughly 30% lower cost per finding. It arrives as AI-generated code inflates enterprise vulnerability backlogs. Available to enterprise customers now.
