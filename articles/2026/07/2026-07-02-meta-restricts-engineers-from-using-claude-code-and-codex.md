# Meta restricts engineers from using Claude Code and Codex

Internal documents reported by The Information indicate Meta has placed strict limits on how engineers in its applied-AI division may use Anthropic's Claude Code and OpenAI's Codex, citing concern about inadvertent distillation of rival models into Meta's own training pipeline. The move signals how seriously frontier labs and their large customers now treat model-to-model knowledge leakage. Single-sourced; treat as reported rather than confirmed.
