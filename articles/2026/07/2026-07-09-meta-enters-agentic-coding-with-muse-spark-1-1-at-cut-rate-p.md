# Meta enters agentic coding with Muse Spark 1.1 at cut-rate pricing

Meta launched Muse Spark 1.1, a multimodal model built for agentic coding, bug fixing, and large code migrations, at roughly $1.25 per million input tokens and $4.25 per million output tokens. The pricing undercuts several rivals and makes Meta a credible option for cost-sensitive enterprise workloads. The launch confirms agentic software engineering is now a core battleground for every major lab.
