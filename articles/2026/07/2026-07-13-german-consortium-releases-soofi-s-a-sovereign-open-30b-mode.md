# German consortium releases Soofi S, a sovereign open 30B model

Soofi S 30B-A3B activates 3.2B of 31.6B parameters per token and tops fully open models on German and English benchmarks. Trained on Deutsche Telekom's Munich cloud using ~512 Nvidia B200 GPUs with a hybrid Mamba-Transformer architecture claiming ~8× throughput vs comparable dense models. A deliberate European sovereignty play.
