# China warns of a security backdoor in Anthropic's Claude Code; Alibaba bans it internally

China's National Vulnerability Database warned that Claude Code versions 2.1.91 through 2.1.196 contain a backdoor that could transmit user data without consent. Alibaba is banning internal Claude Code use from July 10. Anthropic said the behavior was part of a March anti-distillation experiment being rolled back.
