# Stanford proposes constrained decoding for diffusion language models

Stanford researchers proposed an efficient method for constrained decoding in diffusion language models, enabling structured outputs such as function calls, SQL, planning formats, and mathematical constraints. If diffusion LLMs gain traction for parallel generation, this addresses a key production blocker.
