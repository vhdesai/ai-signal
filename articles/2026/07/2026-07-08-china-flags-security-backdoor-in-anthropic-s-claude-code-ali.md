# China Flags "Security Backdoor" in Anthropic's Claude Code; Alibaba Bans It Internally

China's MIIT warned that Claude Code versions 2.1.91–2.1.196 can transmit user data to remote servers without consent, urging users to uninstall or upgrade. Alibaba banned it internally from July 10. Anthropic characterized the data collection as a March anti-abuse experiment being rolled back. The dispute escalates the US–China standoff around model distillation.
