# Security teardown says xAI's Grok Build CLI uploads entire repos, including secrets

Grok Build CLI v0.2.93 transmits the full repository to a GCS bucket — not just agent-read files. A 12 GB repo generated 5.1 GB of uploads with .env sent verbatim; opt-out reportedly ineffective. Enterprises piloting agentic coding tools should audit egress before wider deployment.
