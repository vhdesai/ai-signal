# Microsoft Research Introduces Flint, an Open-Source Visualization Language for Agents

Microsoft Research released Flint, an open-source visualization language that lets AI agents generate expressive charts from compact, human-editable specifications. It targets agent-authored data visualization as analytics work shifts to autonomous systems — a middle path between terse chart specs and hand-tuned custom code.
