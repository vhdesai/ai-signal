# Ollama Raises $65M Series B as Local-AI Tooling Scales to ~9M Users

Open-source local-inference tool Ollama raised $65M led by Theory Ventures (total funding: $88M, ~9M users, 176K GitHub stars). It monetizes via hosted "neocloud" billed on GPU time. The round signals investor conviction that developer-owned, local/edge inference is a lasting market alongside frontier APIs.
