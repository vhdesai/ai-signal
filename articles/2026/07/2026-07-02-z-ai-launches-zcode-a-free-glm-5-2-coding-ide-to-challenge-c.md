# Z.ai launches ZCode, a free GLM-5.2 coding IDE, to challenge Cursor and Claude Code

Beijing-based Z.ai (formerly Zhipu AI) released ZCode, a free desktop "Agentic Development Environment" built for its GLM-5.2 model, available on macOS, Windows, and Linux with bring-your-own-key support. It competes head-on with Cursor, Claude Code, GitHub Copilot, and Google's Antigravity, pairing a capable open-weight Chinese model with a free first-party tool. The launch intensifies price and distribution pressure in AI coding — and, per VentureBeat, sharpens the geopolitical calculus enterprises face when standardizing developer tooling.
