# Security concern: Grok Build (xAI) uploads entire Git repositories to xAI storage

A report surfaced that xAI’s Grok Build agentic coding CLI uploads whole Git repositories to xAI storage rather than only the files it needs to read — raising data-exposure and IP concerns for developers using the tool.
