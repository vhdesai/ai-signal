# OpenAI launches ChatGPT Work, an agent for multi-hour enterprise tasks

Alongside GPT‑5.6, OpenAI unveiled ChatGPT Work, a GPT‑5.6/Codex-powered agent that autonomously handles multi-hour, multi-step workflows across apps and files via a Unified Plugins Directory spanning Slack, Teams, Google Drive, SharePoint, and Salesforce. It merges the standalone Codex app into a new ChatGPT desktop app and adds a hosted "Sites" feature. It is available immediately for Pro, Enterprise, and Edu, with Plus and Business to follow.
