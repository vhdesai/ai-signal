# LlamaIndex published legal-kb, a public reference application that gives AI agents filesystem-style access to a…

LlamaIndex published legal-kb, a public reference application that gives AI agents filesystem-style access to a document knowledge base built on Index v2. It exposes retrieve (hybrid semantic search), find, read, and grep as agent tools, with per-file versioning and visual citations. It is a compact but useful template for teams building document-grounded agents.
