# Meta Launches Muse Image, First Model from Meta Superintelligence Labs

Meta unveiled Muse Image, an agentic image-generation model that uses tools such as code execution and web search to refine its own outputs, similar to OpenAI’s GPT Image 2. Available free in the Meta AI app, Instagram Stories, and WhatsApp, it is the first model to ship from Meta’s Superintelligence Labs. The launch drew immediate privacy scrutiny over a feature that lets users AI-manipulate other Instagram users’ public photos, raising consent and data-use questions.
