# Carnegie Mellon releases Robot I/O, open-source infrastructure to move AI between robots

CMU released Robot I/O, an open-source Python framework for robot control, data collection, teleoperation, and VLA deployment across single-arm, bimanual, and humanoid platforms. The framework addresses a practical bottleneck: moving robot AI across hardware without rebuilding the stack each time.
