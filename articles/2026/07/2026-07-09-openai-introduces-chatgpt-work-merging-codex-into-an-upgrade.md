# OpenAI introduces ChatGPT Work, merging Codex into an upgraded desktop app

Alongside GPT-5.6, OpenAI launched ChatGPT Work — a cross-platform enterprise agent (web, mobile, desktop) that combines ChatGPT with the Codex coding tool to produce documents, spreadsheets, decks, reports, websites, and software from a single interface. The desktop app was rebuilt to bundle Chat, Work, and Codex, and OpenAI added a hosted-sites feature for paid users. The product directly targets Anthropic's Claude Cowork, Microsoft 365 Copilot, Google's Gemini for Workspace, and Amazon Q Business.
