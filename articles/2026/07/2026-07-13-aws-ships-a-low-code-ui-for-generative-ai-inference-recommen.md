# AWS ships a low-code UI for generative-AI inference recommendations in SageMaker AI

Amazon launched a UI in SageMaker AI Studio that walks teams through preset use-case profiles, visual benchmark comparisons, and one-click deployment for optimized inference — removing the need to hand-tune parameters. A smaller but firmly in-window enterprise tooling update.
