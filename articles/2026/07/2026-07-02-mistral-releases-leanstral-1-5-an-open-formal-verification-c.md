# Mistral releases Leanstral 1.5, an open formal-verification code agent for Lean 4

Mistral released Leanstral 1.5, an Apache-2.0-licensed 119B mixture-of-experts model (only ~6B active parameters) built to generate and check mathematical proofs and verify code in the Lean 4 proof language. It saturates the miniF2F benchmark and solves 587 of 672 PutnamBench problems at roughly $4 per problem — versus an estimated $300+ for a leading competitor — and reportedly surfaced five previously unreported bugs across open-source repositories. The practical angle for enterprises: as coding agents write more production code, low-cost formal verification can sit downstream of code generation to prove critical components correct rather than merely test them. It is also a notable self-hostable, "sovereign" option for regulated engineering teams.
