# Choosing the right AI-agent memory strategy: a decision-tree approach

This practitioner guide frames AI-agent memory design as a five-question decision tree applied per category of information rather than per agent. It distinguishes four memory types — working, semantic, episodic, and procedural — and maps each to concrete implementations such as conversation buffers, knowledge graphs, event logs, and distilled procedural stores. It closes with a pitfalls-and-fixes table for common failure modes.
