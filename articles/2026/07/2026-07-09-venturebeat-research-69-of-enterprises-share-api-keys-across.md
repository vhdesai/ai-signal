# VentureBeat research: 69% of enterprises share API keys across AI agents, creating blast-radius risk

A VentureBeat Pulse survey of 107 enterprises found 69% run AI agents with shared credentials, so a single compromised agent inherits the combined permissions of every workflow the key touches — and shared accounts erase the audit trail of which agent did what. The report ties the exposure to a wave of security consolidation targeting the agent-identity layer, including Palo Alto Networks' $21.1B CyberArk acquisition and additional CrowdStrike and Cisco bets. For executives scaling agent fleets, credential isolation is emerging as a near-term governance gap.
