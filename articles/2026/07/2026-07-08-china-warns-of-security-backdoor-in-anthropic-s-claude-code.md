# China Warns of "Security Backdoor" in Anthropic's Claude Code; Alibaba Bans It

China's National Vulnerability Database warned that Claude Code v2.1.91–2.1.196 contain a backdoor that could transmit user data without consent. Alibaba is banning internal Claude Code from July 10. Anthropic said it was a March anti-distillation experiment being rolled back. The warning lands amid intensifying US–China AI tensions.
