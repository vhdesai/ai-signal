# Anthropic's pre-release "Fable 5" rewrites the Bun runtime from Zig to Rust — ~1M lines in 11 days

Bun creator Jarred Sumner (now at Anthropic) used a pre-release Claude "Fable 5" model to port the Bun JavaScript runtime's ~960,000-line codebase from Zig to Rust, running about 64 parallel instances over 11 days at roughly $165,000 in API cost — work he estimated would take engineers about a year. The Rust port reportedly reached ~99.8% test compatibility on Linux x64. It is one of the largest public demonstrations of AI-driven software migration to date, though achieved with privileged pre-release access that limits third-party replicability.
