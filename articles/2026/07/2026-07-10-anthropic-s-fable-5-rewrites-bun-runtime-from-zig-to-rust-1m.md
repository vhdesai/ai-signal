# Anthropic's "Fable 5" Rewrites Bun Runtime from Zig to Rust — ~1M Lines in 11 Days

Bun creator Jarred Sumner used a pre-release Claude "Fable 5" to port Bun's ~960K-line codebase from Zig to Rust, running ~64 parallel instances over 11 days at ~$165K. The Rust port reached ~99.8% test compatibility. One of the largest public demonstrations of AI-driven software migration — though achieved with privileged pre-release access that limits third-party replicability.
