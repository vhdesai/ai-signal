# OpenAI reportedly cuts inference costs by more than 50% through software alone

OpenAI engineers developed optimization techniques — a combination of quantization, batching, model routing, and (most consequentially) key-value caching — that cut model-serving costs by more than 50%, with no new hardware. At one point, logged-out ChatGPT traffic was served by only a couple hundred GPUs. If it holds at fleet scale, this is a structural repricing of AI unit economics and pressure on every rival's API margins; customer-facing price changes have not yet been announced.
