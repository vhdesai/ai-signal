# Microsoft recaps June's Microsoft 365 Copilot feature drop, led by Copilot Cowork GA

A July 1 roundup details the Microsoft 365 Copilot updates shipped in June, headlined by the general availability of Copilot Cowork, which completes full business tasks rather than responding to single prompts. Other June additions included GPT-5.5 Thinking model selection, Anthropic model support for visual work, browser automation, mobile support, deep citations, and a Regenerate button. Note: this is a monthly recap published in-window, not a single new July 1 launch. https://windowsreport.com/microsoft-365-copilot-received-these-new-features-in-june-2026/
