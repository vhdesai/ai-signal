# Terence Tao Resurrects Two Dozen 1999 Math Applets Using an AI Coding Agent

Fields Medalist Terence Tao ported ~24 defunct Java 1.0 applets to JavaScript in hours using an AI coding agent — an expert-validated data point on agentic code migration directly relevant to enterprise modernization. The agent also surfaced one new bug alongside two pre-existing ones.
