# China says it found “security backdoor” vulnerabilities in Anthropic’s Claude Code

China’s National Vulnerability Database, under the MIIT, warned that several versions of Anthropic’s Claude Code (v2.1.91–2.1.196, released April–June) can transmit user location and identity data to remote servers without consent via a built-in monitoring mechanism, urging users to uninstall or upgrade. The alert follows Alibaba’s move to ban the tool internally from July 10. An Anthropic engineer said on X the code was part of a March anti-abuse / anti-distillation experiment being rolled back. The dispute escalates an ongoing US–China standoff in which Anthropic has accused Chinese labs of illicit model distillation.
