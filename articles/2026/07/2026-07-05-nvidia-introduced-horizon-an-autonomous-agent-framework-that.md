# NVIDIA introduced HORIZON, an autonomous agent framework that treats each register-transfer-level (RTL) hardware-design…

NVIDIA introduced HORIZON, an autonomous agent framework that treats each register-transfer-level (RTL) hardware-design problem as a versioned Git repository the agent evolves on its own. NVIDIA reports the system reached 100% completion across its benchmark suite. The release signals continued momentum toward AI agents that design silicon, not just software.
