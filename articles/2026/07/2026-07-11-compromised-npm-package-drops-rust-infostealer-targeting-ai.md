# Compromised npm Package Drops Rust Infostealer Targeting AI Dev Tools

The official jscrambler npm package was compromised; a preinstall hook drops a cross-platform Rust infostealer targeting cloud credentials, browser data, and crypto wallets. It also harvests config files of Claude Desktop, Cursor, Windsurf, VS Code, and Zed. No fix was available as of July 11 — immediate heads-up to engineering teams warranted.
