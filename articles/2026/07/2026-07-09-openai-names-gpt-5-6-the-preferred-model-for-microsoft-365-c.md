# OpenAI Names GPT-5.6 the "Preferred Model" for Microsoft 365 Copilot

OpenAI said GPT-5.6 will be the preferred model across M365 Copilot — Word, Excel, PowerPoint, Chat, and Cowork. The disclosure follows Bloomberg's earlier-week report that Microsoft was substituting MAI models to cut costs. Microsoft surfaces the model as "Quick Response" / "Think Deeper" rather than Sol/Terra/Luna. The competing narrative — Microsoft pulling away vs. OpenAI reinforcing the partnership — remains unresolved.
