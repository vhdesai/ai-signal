# LlamaIndex ships "legal-kb," an agentic retrieval reference app on Index v2

LlamaIndex released legal-kb, a public reference app that gives AI agents filesystem-style access to a document knowledge base on Index v2, exposing retrieve (hybrid semantic search), find, read, and grep as tools with automatic per-file versioning and visual citations. The stack pairs TanStack Start, AI SDK 6 (ToolLoopAgent), Prisma, and WorkOS. It's a useful blueprint for teams building agentic document-retrieval workflows over enterprise content.
