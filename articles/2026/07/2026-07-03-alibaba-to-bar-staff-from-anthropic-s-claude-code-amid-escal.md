# Alibaba to bar staff from Anthropic's Claude Code amid escalating distillation feud

Alibaba will prohibit employees from using Claude Code inside workplace environments effective July 10, citing an alleged "backdoor" — first reported by China's Yicai and corroborated by Reuters. The move follows researchers' claims that Claude Code contained steganographic markers flagging users in Chinese time zones (which Anthropic called an experiment), weeks after Anthropic accused operators tied to Alibaba's Qwen lab of the largest known distillation campaign against Claude. The tit-for-tat marks a sharp escalation in US–China AI decoupling, now playing out at the level of individual developer tools — a rising vendor-trust and jurisdictional risk for enterprises using cross-border AI tooling.
