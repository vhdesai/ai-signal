# NVIDIA Frames Vera CPU as “Max Single-Threaded CPU at Scale”; Teases Next-Gen ‘Rigel’ Cores

NVIDIA published a blog framing its Vera CPU as a new category — “max single-threaded CPU at scale” — arguing that for agentic systems the CPU sits on the critical path for reasoning, response time, and learning, a contrast to the usual parallel-throughput framing. Tom’s Hardware’s coverage notes NVIDIA also teased next-generation ‘Rigel’ Arm CPU cores. The post accompanies NVIDIA’s Vera Rubin platform push into agentic and physical-AI infrastructure.
