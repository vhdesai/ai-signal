# Anthropic Expands Claude Cowork to Web and Mobile, Adds Microsoft 365 Write Tools

Anthropic extended Claude Cowork to the web and iPhone, with sessions now running remotely so work persists across devices. The M365 connector gained write tools — Claude can now draft email, manage calendar events, and create or update OneDrive and SharePoint files. Teams remains read-only; a Microsoft Entra admin must consent before the tools are enabled.
