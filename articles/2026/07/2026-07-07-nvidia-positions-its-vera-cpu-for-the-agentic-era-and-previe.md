# Nvidia positions its Vera CPU for the agentic era and previews next-gen Rosa CPU / Rigel core

In a positioning post, Nvidia framed its 88-core Arm “Olympus”-based Vera CPU as a new category — the “max single-threaded CPU at scale” — arguing that per-core speed, not core count, governs latency inside agent loops and protects expensive GPU utilization. It cited Perplexity testing Vera at ~1.5x faster than x86 on real coding workflows and, notably, previewed the roadmap: a next-generation Rosa CPU built on a new Arm v9.2 “Rigel” core. The message underscores Nvidia’s push beyond GPUs into the data-center CPU socket as agentic workloads reshape infrastructure economics.
