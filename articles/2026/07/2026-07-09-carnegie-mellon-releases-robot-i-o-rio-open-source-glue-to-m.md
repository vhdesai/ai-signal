# Carnegie Mellon Releases Robot I/O (RIO), Open-Source Glue to Move AI Between Robots

CMU released RIO, an open-source Python framework providing a unified interface for robot control, data collection, teleoperation, and VLA deployment across single-arm, bimanual, and humanoid platforms. One intern went from unboxing a robotic arm to teleoperating it in ~2 hours. Accepted to RSS 2026.
