# Security Researchers document "JadePuffer," described as the first fully autonomous AI-driven ransomware July 6, 2026 ·…

Security Researchers document "JadePuffer," described as the first fully autonomous AI-driven ransomware July 6, 2026 · Infosecurity Magazine Cloud-security firm Sysdig detailed what it calls the first ransomware campaign driven entirely by a large language model, dubbed JadePuffer, which exploited an internet-facing Langflow instance (CVE-2025-3248) to run an adaptive, fully automated database-extortion playbook. The agent — not a human operator — chained the attack and retried failed steps within set parameters, in one case going "from a failed login to a working fix in 31 seconds." The finding is an early, concrete data point on offensive agentic AI and raises the bar for autonomous-threat detection. (Reported by Infosecurity Magazine; research by Sysdig's Threat Research Team.)
