# "Slopsquatting" emerges as an AI-coding supply-chain threat

VentureBeat detailed "slopsquatting," a supply-chain attack that weaponizes LLM hallucinations: attackers register the fictitious package names that AI coding assistants tend to invent, then wait for those tools to recommend and install the malicious code. As developers lean harder on AI assistants, the technique can hand attackers a foothold in software from day one. It is part of a fast-growing class of agentic-coding risks that security teams will need to fold into build pipelines and dependency review.
