# SpaceXAI Sets Grok 4.5 Public Release for Thursday; Joint Cursor Model Imminent

Elon Musk said SpaceXAI's Grok 4.5 will become publicly available Thursday, describing it as an "Opus-class" model built on xAI's new V9 foundation. Separately, SpaceXAI and Cursor plan to ship their first jointly developed frontier model as soon as today, trained on xAI's Colossus supercomputer using Cursor's developer-session data. The model targets Anthropic's Opus 4.8 and OpenAI's GPT-5.5.
