# Nvidia Positions Vera CPU for the Agentic Era, Previews Next-Gen Rosa CPU

Nvidia framed its 88-core Arm "Olympus"-based Vera CPU as a new category for agentic workloads, arguing that per-core speed governs latency inside agent loops. Perplexity testing showed ~1.5× faster than x86 on real coding workflows. Nvidia previewed the next-generation Rosa CPU built on a new Arm v9.2 "Rigel" core — underscoring Nvidia's push beyond GPUs into the data-center CPU socket.
