# Prismata: Confining Cross-Site Prompt Injection in Web Agents

UC Berkeley researchers introduced Prismata, a system-level defense against cross-site prompt injection attacks in web agents. The work is relevant to enterprise AI deployment because web agents will increasingly operate across untrusted content and authenticated actions; Prismata’s approach uses DOM-derived trust and least-privilege enforcement to reduce attack success while preserving useful task execution.
