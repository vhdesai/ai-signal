# Meta enters the coding-model race with Muse Spark 1.1

Meta unveiled Muse Spark 1.1, a multimodal reasoning model built for agentic tasks and software development, alongside a public preview of a new Meta Model API. Meta calls it its "strongest model for agentic and coding work yet," with gains in tool use, computer use, and coding — an explicit move onto the turf OpenAI and Anthropic have been contesting. Landing the same week as GPT‑5.6 and xAI's Grok 4.5, it confirms that agentic coding is now the primary battleground among frontier labs.
