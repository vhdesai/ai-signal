# Claude Fable 5 rewrites Bun runtime from Zig to Rust in a large-scale AI migration demo

Bun creator Jarred Sumner used a pre-release Claude Fable 5 model to port the Bun JavaScript runtime's roughly 960,000-line codebase from Zig to Rust. The effort reportedly ran about 64 parallel model instances over 11 days at roughly $165,000 in API cost and reached about 99.8% Linux x64 test compatibility.
