# Cognition launches Devin Security Swarm to attack the vulnerability backlog

Cognition, the lab behind the Devin coding agent, launched Devin Security Swarm — a system that scans large codebases for vulnerabilities, validates exploitability at runtime in sandboxes, and opens remediation pull requests using an "agentic MapReduce" architecture. The company frames it as a response to AI-generated code, now roughly 42% of enterprise code, which it says is pushing monthly security findings from about 1,000 to more than 10,000 within six months. It marks a notable pivot of agentic AI from writing code to securing it at scale.
