# Stanford introduces TRACE — a capability-targeted agentic training system

Stanford researchers unveiled TRACE, which turns an agent’s recurring failures into synthetic reinforcement-learning environments, then trains directly against the specific capability gaps causing those failures. The approach is aimed at making agentic LLMs more reliable on long-horizon, multi-step tasks — a core blocker for enterprise agent deployment.
