# OpenAI ships ChatGPT Work, an enterprise agent spanning Slack, Teams, Drive, and SharePoint

OpenAI unveiled ChatGPT Work, a GPT-5.6-powered enterprise agent that connects to workplace apps and files through a Unified Plugins Directory. It can produce finished documents, presentations, reports, websites, and other outputs through scheduled background workflows, while folding Codex into a single desktop super app. The competitive axis is shifting from model quality alone toward who owns the everyday interface for AI-assisted work.
