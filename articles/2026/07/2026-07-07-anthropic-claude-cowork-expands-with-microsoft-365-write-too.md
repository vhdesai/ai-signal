# Anthropic Claude Cowork Expands with Microsoft 365 Write Tools

Anthropic expanded its Claude Cowork platform with new write tools integrated into Microsoft 365 applications. The expansion allows Claude to directly draft, edit, and format documents within Word, Excel, and PowerPoint — deepening Anthropic's enterprise integration strategy and competing directly with Microsoft's own Copilot features.
