# OpenAI launches ChatGPT Work, an enterprise agent spanning Slack, Teams, Drive and SharePoint

OpenAI unveiled ChatGPT Work, a GPT-5.6-powered enterprise offering that connects to workplace apps and files to produce finished documents, spreadsheets, presentations, reports, and websites via scheduled, background workflows. It positions OpenAI directly against Anthropic's Claude Cowork and Microsoft's Copilot. The signal for leaders: the next competitive axis is not model quality alone but ownership of the everyday interface for AI-assisted work.
