# OpenAI introduces ChatGPT Work, merging Codex into an enterprise agent platform

OpenAI unveiled ChatGPT Work, a GPT-5.6/Codex-powered platform for multi-hour workflows across workplace apps. It connects through a Unified Plugins Directory with Slack, Teams, Google Drive, SharePoint, and Salesforce, and bundles Chat, Work, Codex, and hosted Sites into the desktop app. The move pushes OpenAI toward an enterprise super-app spanning writing, coding, research, and automation.
