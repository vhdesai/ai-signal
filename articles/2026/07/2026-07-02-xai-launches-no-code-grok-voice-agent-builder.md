# xAI launches no-code Grok Voice Agent Builder

xAI released the Grok Voice Agent Builder in beta, a no-code tool it says lets users stand up production voice agents in under two minutes, bundling telephony, document retrieval, tool-calling and guardrails into a single speech-to-speech interface. The product targets customer support, sales and personal-assistant workflows, with browser-based testing and enterprise integrations. It pushes xAI directly into the fast-growing voice-agent category alongside OpenAI and specialist startups. Performance benchmarks are self-reported and unverified.
