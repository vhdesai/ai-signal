# Carnegie Mellon helps launch FLARE-AI, an open-source pipeline for coordinated AI-flaw disclosure

Researchers at Carnegie Mellon University’s Software Engineering Institute helped develop FLARE-AI, a new open-source platform for reporting AI flaws and coordinating responses across developers, vendors and government agencies. The effort addresses a structural gap: a vulnerability found in one model can be silently replicated across the many products built on the same underlying system, yet the field has lacked a formal disclosure pathway.
