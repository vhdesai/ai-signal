# China Warns of “Backdoor” Security Risk in Anthropic’s Claude Code; Alibaba to Ban It from July 10

China’s National Vulnerability Database, tied to the Ministry of Industry and Information Technology, warned that Claude Code versions 2.1.91–2.1.196 contain a “security backdoor” that can transmit user location and identity data to remote servers without consent, urging users to uninstall or upgrade to the latest 2.1.204. The warning escalates U.S.–China tech tensions: Anthropic recently accused Alibaba of “distillation,” and Alibaba has told employees to stop using Claude Code from July 10. A Claude Code engineer earlier characterized the flagged behavior as an anti-abuse and anti-distillation experiment launched in March that was being rolled back.
