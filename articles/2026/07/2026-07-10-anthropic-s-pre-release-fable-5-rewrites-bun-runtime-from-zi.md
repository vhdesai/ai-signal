# Anthropic's pre-release Fable 5 rewrites Bun runtime from Zig to Rust

Bun creator Jarred Sumner used a pre-release Claude “Fable 5” model to port the Bun JavaScript runtime's roughly 960,000-line codebase from Zig to Rust. The migration used about 64 parallel model instances over 11 days at an estimated $165,000 in API cost, and the Rust port reportedly reached about 99.8% test compatibility on Linux x64.
